Module Overview

This module focuses on custom WordPress theme development. You'll learn how to create themes from scratch, understand the WordPress template hierarchy in depth, and implement advanced theming techniques. By the end of this module, you'll be able to build professional, custom WordPress themes that meet specific design and functionality requirements.

Duration: 2 weeks (10 sessions of 3 hours each)

Prerequisites: Modules 1-4: Web Fundamentals, PHP, MySQL, and WordPress Fundamentals

Week 1: Theme Fundamentals and Structure

Session 1: Theme Development Introduction

  • WordPress theme architecture overview
  • Theme requirements and standards
  • Theme development environment setup
  • Creating a basic theme structure
    • Required files (style.css, index.php)
    • Theme metadata
    • Theme screenshot
  • Theme testing and validation
  • Homework: Create a minimal WordPress theme with required files

Session 2: WordPress Template Hierarchy

  • Template hierarchy in depth
  • Main template files:
    • index.php
    • header.php and footer.php
    • sidebar.php
    • single.php and page.php
    • archive.php and category.php
    • search.php and 404.php
  • Template parts and includes
  • get_template_part() function
  • Homework: Expand your theme with template files following the hierarchy

Session 3: The Loop and WordPress Content

  • Understanding The Loop in detail
  • Post data and template tags
  • Customizing The Loop
  • Multiple loops on a single page
  • Loop alternatives (WP_Query, get_posts)
  • Pagination techniques
  • Homework: Implement custom loops in your theme templates

Session 4: Theme Functions and hooks

  • Understanding functions.php
  • Theme setup functions
  • Enqueuing styles and scripts
  • WordPress action hooks
    • wp_head and wp_footer
    • init and after_setup_theme
    • wp_enqueue_scripts
  • WordPress filter hooks
  • Creating custom hooks
  • Homework: Implement theme support features and create custom hooks

Session 5: Theme Customization Features

  • WordPress Customizer API
  • Adding custom Customizer sections and controls
  • Implementing theme options
  • Live preview with the Customizer
  • Color schemes and typography options
  • Selective refresh
  • Homework: Add Customizer options to your theme

Week 2: Advanced Theme Development

Session 6: Navigation and Widgets

  • Registering navigation menus
  • Implementing menus in themes
  • Walker classes for custom menu output
  • Registering widget areas
  • Adding widgets to themes
  • Creating custom widgets
  • Homework: Implement custom navigation menus and widget areas

Session 7: Custom Post Types and Taxonomies

  • Understanding Custom Post Types (CPTs)
  • Registering CPTs
  • Custom Post Type templates
  • Custom taxonomies
  • Term metadata
  • Displaying custom content in themes
  • Homework: Create a custom post type with taxonomies and display templates

Session 8: Theme Assets and Frontend

  • Asset organization in themes
  • Responsive design implementation
  • Using Sass in theme development
  • JavaScript in WordPress themes
  • Working with jQuery in WordPress
  • Using third-party libraries
  • Asset optimization and best practices
  • Homework: Implement responsive design and optimize theme assets

Session 9: Advanced Theme Features

  • Theme internationalization (i18n)
  • Translation-ready themes
  • Accessibility considerations
  • SEO best practices in themes
  • Schema markup
  • Theme security best practices
  • Theme performance optimization
  • Homework: Make your theme translation-ready and implement accessibility features

Session 10: Theme Project and Distribution

  • Theme development workflow
  • Version control for themes
  • Theme testing checklist
  • Theme documentation
  • Packaging themes for distribution
  • WordPress.org theme requirements
  • Theme marketplaces
  • Final Project: Complete a fully functional custom WordPress theme

Assessment

  • Daily homework assignments: 30%
  • Participation and in-class exercises: 20%

Important Notes

This module requires consistent practice and coding. Students are encouraged to work on their theme development projects daily, even outside of class hours. Theme development is both technical and creative - students should consider both aspects when working on their projects.

The final project will involve creating a complete custom theme that could potentially be used in a professional portfolio.

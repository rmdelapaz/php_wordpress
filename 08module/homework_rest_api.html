üéØ Assignment Overview

Create a "Task Manager API" plugin that provides a complete REST API for managing tasks. Your plugin should demonstrate mastery of REST API concepts including custom endpoints, authentication, and extending existing endpoints.

Plugin Concept

Build a task management system with:

  • Custom post type for tasks
  • REST API endpoints for CRUD operations
  • User authentication and permissions
  • Frontend interface using REST API
  • Integration with existing WordPress endpoints

üìã Requirements

Core Features (Required)

  • Custom post type "task" with REST API support
  • Custom REST API endpoints for tasks
  • Authentication using nonces or application passwords
  • Permission checks for all operations
  • Extend posts endpoint with task-related data
  • Frontend interface using JavaScript/REST API
  • Error handling and validation
  • API documentation

API Endpoints to Implement

  • GET /wp-json/taskmanager/v1/tasks - List all tasks
  • GET /wp-json/taskmanager/v1/tasks/{id} - Get single task
  • POST /wp-json/taskmanager/v1/tasks - Create task
  • PUT /wp-json/taskmanager/v1/tasks/{id} - Update task
  • DELETE /wp-json/taskmanager/v1/tasks/{id} - Delete task
  • GET /wp-json/taskmanager/v1/tasks/stats - Task statistics
  • POST /wp-json/taskmanager/v1/tasks/{id}/complete - Mark complete

    public function register_routes() {
        register_rest_route('taskmanager/v1', '/tasks', array(
            array(
                'methods' => WP_REST_Server::READABLE,
                'callback' => array($this, 'get_tasks'),
                'permission_callback' => '__return_true',
            ),
            array(
                'methods' => WP_REST_Server::CREATABLE,
                'callback' => array($this, 'create_task'),
                'permission_callback' => array($this, 'create_permission'),

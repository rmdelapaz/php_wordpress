Introduction to CSS and Its Purpose

Duration: 45 minutes
Module 1, Session 3

Learning Objectives

  • Understand what CSS is and why it's essential
  • Learn how CSS works with HTML
  • Understand the separation of concerns principle
  • Learn basic CSS syntax and structure
  • Explore different ways to implement CSS
  • Understand the cascade and specificity

What is CSS?

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of HTML documents. While HTML provides structure and meaning to content, CSS controls how that content looks and is laid out on the page.

💡
Think of it this way
If HTML is the skeleton of a website, CSS is the skin, clothes, makeup, and accessories that make it visually appealing and unique.

Why CSS is Essential

Before CSS

  • Styling mixed with content
  • Inconsistent appearance
  • Difficult to maintain
  • Limited styling options
  • Repetitive code

With CSS

  • Clean separation of concerns
  • Consistent styling
  • Easy to update
  • Rich styling capabilities
  • Reusable styles

How CSS Works

CSS works by selecting HTML elements and applying styles to them. The browser combines your HTML structure with CSS styles to render the final page.

The Rendering Process

  1. Browser loads the HTML document
  2. Browser parses HTML to create the DOM (Document Object Model)
  3. Browser loads and parses CSS
  4. Browser creates the CSSOM (CSS Object Model)
  5. Browser combines DOM and CSSOM to create the render tree
  6. Browser calculates layout and paints pixels to screen

Basic CSS Syntax

CSS consists of rules that target HTML elements and apply styles to them:

selector {
    property: value;
    another-property: another-value;
}

Example CSS Rules

/* Element selector */
h1 {
    color: blue;
    font-size: 32px;
}

/* Class selector */
.highlight {
    background-color: yellow;
    padding: 5px;
}

/* ID selector */
#header {
    background-color: #f0f0f0;
    padding: 20px;
}

/* Multiple selectors */
h1, h2, h3 {
    font-family: Arial, sans-serif;
}

Ways to Implement CSS

1. Inline CSS

Applied directly to HTML elements using the style attribute:

<p style="color: red; font-size: 20px;">This is red text.</p>
⚠️
Not Recommended
Inline styles mix presentation with content and are difficult to maintain. Use only for quick testing or one-off styling.

2. Internal CSS

Placed in the <head> section using <style> tags:

<head>
    <style>
        body {
            background-color: #f0f0f0;
        }
        h1 {
            color: navy;
        }
    </style>
</head>

3. External CSS (Recommended)

Linked as a separate file:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

Best Practice

External CSS files are the best approach because they:

  • Keep HTML and CSS separate
  • Can be cached by browsers for faster loading
  • Allow one CSS file to style multiple HTML pages
  • Are easier to maintain and update

The Cascade

The "C" in CSS stands for "Cascading." When multiple styles apply to the same element, the cascade determines which one wins:

Cascade Order (from lowest to highest priority)

  1. Browser default styles
  2. External stylesheets
  3. Internal styles (in <style> tag)
  4. Inline styles
  5. Styles marked with !important

Specificity

When styles have the same priority level, specificity determines the winner:

Selector Type Specificity Value Example
Element 0-0-1 p
Class 0-1-0 .highlight
ID 1-0-0 #header
Inline 1-0-0-0 style="..."

Common CSS Properties

Text Properties

  • color - Text color
  • font-size - Text size
  • font-family - Font typeface
  • font-weight - Bold/normal
  • text-align - Alignment

Box Properties

  • width - Element width
  • height - Element height
  • padding - Inner spacing
  • margin - Outer spacing
  • border - Element border

Your First CSS

Let's style a simple HTML page:

HTML:

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Welcome to My Site</h1>
    <p class="intro">This is an introduction paragraph.</p>
    <p>This is regular text.</p>
</body>
</html>

CSS (style.css):

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 20px;
    background-color: #f4f4f4;
}

h1 {
    color: #333;
    border-bottom: 3px solid #333;
    padding-bottom: 10px;
}

.intro {
    font-size: 1.2em;
    color: #666;
    font-style: italic;
}

p {
    margin-bottom: 15px;
}

WordPress Context

CSS in WordPress

In WordPress development, CSS is crucial for:

  • Theme Development: Creating custom WordPress themes
  • Customization: Modifying existing themes
  • Responsive Design: Making sites mobile-friendly
  • Plugin Styling: Styling plugin output

WordPress themes typically use:

/* WordPress Theme style.css */
/*
Theme Name: My Custom Theme
Author: Your Name
Description: A custom WordPress theme
Version: 1.0
*/

/* Your CSS starts here */
body {
    font-family: 'Open Sans', sans-serif;
}

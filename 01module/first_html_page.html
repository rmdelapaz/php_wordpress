Creating Your First HTML Page

Duration: 45 minutes
Module 1, Session 1

Learning Objectives

  • Understand the basic structure of an HTML document
  • Learn about DOCTYPE and its importance
  • Create a complete HTML page from scratch
  • Add content using basic HTML tags
  • View and test your page in a browser
  • Understand HTML validation

Introduction

Today's the day you become a web developer! We're going to create your very first HTML page from scratch. By the end of this lesson, you'll have a working webpage that you can view in your browser.

🎉
Exciting Milestone!
Creating your first HTML page is a significant milestone. Every web developer remembers their first "Hello, World!" webpage. Today, you'll create yours!

What is HTML?

HTML (HyperText Markup Language) is the foundation of every webpage. It provides structure and meaning to content.

Key Concepts

  • HTML is NOT a programming language - it's a markup language
  • HTML uses tags to mark up content
  • Tags usually come in pairs: opening and closing tags
  • HTML is case-insensitive, but we use lowercase by convention
  • HTML files have the .html extension

Step 1: Create Your Project Folder

  1. Open VS Code
  2. Create a new folder called my-first-webpage in your module1 folder
  3. Open this folder in VS Code (File → Open Folder)
  4. Create a new file called index.html
ℹ️
Why index.html?
Web servers automatically look for a file named "index.html" when someone visits a website. It's the default homepage file.

Step 2: The Basic HTML Structure

Every HTML document follows this basic structure. Type this into your index.html file:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page!</p>
</body>
</html>

Let's Break This Down

Element Purpose Details
<!DOCTYPE html> Document type declaration Tells the browser this is HTML5
<html> Root element Contains all other elements
lang="en" Language attribute Helps screen readers and search engines
<head> Metadata container Contains info about the page
<meta charset="UTF-8"> Character encoding Ensures proper text display
<meta name="viewport"> Responsive design Makes page mobile-friendly
<title> Page title Shows in browser tab
<body> Content container Everything visible goes here

Step 3: Viewing Your Page

Now let's see your creation in action!

Method 1: Live Server (Recommended)

  1. Right-click on index.html in VS Code
  2. Select "Open with Live Server"
  3. Your browser will open automatically
  4. Any changes you make will auto-refresh!

Method 2: Direct Opening

  1. Save your file (Ctrl+S)
  2. Find the file in your file explorer
  3. Double-click to open in browser
  4. Refresh manually after changes
✅
Congratulations!
You've just created your first webpage! You should see "Hello, World!" as a large heading and "This is my first HTML page!" as a paragraph.

Step 4: Adding More Content

Let's enhance your page with more content. Replace everything in the <body> section with:

<body>
    <!-- Header Section -->
    <header>
        <h1>Welcome to My Website</h1>
        <p>Created by [Your Name]</p>
    </header>
    
    <!-- Main Content -->
    <main>
        <h2>About Me</h2>
        <p>Hi! I'm learning web development with the PHP WordPress course.</p>
        <p>This is my very first HTML page, and I'm excited to learn more!</p>
        
        <h2>What I'm Learning</h2>
        <ul>
            <li>HTML - The structure of web pages</li>
            <li>CSS - Making things look beautiful</li>
            <li>JavaScript - Adding interactivity</li>
            <li>PHP - Server-side programming</li>
            <li>WordPress - Content management</li>
        </ul>
        
        <h2>My Goals</h2>
        <ol>
            <li>Build a portfolio website</li>
            <li>Create a WordPress theme</li>
            <li>Develop a custom plugin</li>
            <li>Launch a real website</li>
        </ol>
    </main>
    
    <!-- Footer Section -->
    <footer>
        <p>&copy; 2025 My First Website. All rights reserved.</p>
    </footer>
</body>

New Elements Introduced

Structural Elements

  • <header> - Page or section header
  • <main> - Main content area
  • <footer> - Page or section footer
  • <!-- --> - Comments (not visible)

Content Elements

  • <h2> - Second-level heading
  • <ul> - Unordered (bulleted) list
  • <ol> - Ordered (numbered) list
  • <li> - List item
  • &copy; - Copyright symbol ©

Let's make your page more interactive by adding links and an image:

<p>Here are some helpful websites:</p>
<ul>
    <li><a href="https://www.w3schools.com">W3Schools</a> - Web development tutorials</li>
    <li><a href="https://developer.mozilla.org">MDN Web Docs</a> - Comprehensive documentation</li>
    <li><a href="https://wordpress.org">WordPress.org</a> - Official WordPress site</li>

<h2>My Workspace</h2>
<img src="https://via.placeholder.com/600x400" alt="My workspace setup">
<p>This is where the magic happens!</p>

Best Practices for Links and Images

  • Always include the alt attribute for images (accessibility)
  • Use descriptive link text, not "click here"
  • External links should open in the same tab by default
  • Use placeholder images for testing: placeholder.com

Step 6: HTML Validation

It's important to write valid HTML. Let's check your code:

  1. Visit the W3C HTML Validator
  2. Copy your entire HTML code
  3. Paste it into the validator
  4. Click "Check"
  5. Fix any errors or warnings
💡
VS Code Tip
VS Code automatically highlights HTML errors with red squiggly lines. Hover over them to see what's wrong!

Common HTML Mistakes to Avoid

Rookie Mistakes

❌ Wrong ✅ Correct Why It Matters
<h1>Title</h2> <h1>Title</h1> Mismatched tags cause rendering issues
<p>Text<p>More text</p> <p>Text</p><p>More text</p> Unclosed tags break page structure
<img src="pic.jpg"> <img src="pic.jpg" alt="Description"> Missing alt text hurts accessibility
Content outside body All content in body Content must be in the body section

Challenge: Personalize Your Page

Now it's time to make this page truly yours! Try these enhancements:

Easy Challenges

  • Change the page title to include your name
  • Add your favorite quote using <blockquote>
  • Create a list of your hobbies
  • Add more paragraphs about yourself

Medium Challenges

  • Add a navigation menu using <nav>
  • Create a contact section with email
  • Add multiple images in a gallery
  • Use <strong> and <em> for emphasis

Homework Assignment

Create a personal "About Me" webpage that includes:

  1. A proper HTML5 structure with all required elements
  2. At least 3 different heading levels (h1, h2, h3)
  3. At least 2 paragraphs about yourself
  4. One ordered list and one unordered list
  5. At least 3 links to external websites
  6. At least 2 images (can use placeholder images)
  7. A footer with copyright information
  8. HTML comments explaining your code sections
📝
Submission Guidelines
  • Name your file: about-me.html
  • Validate your HTML using W3C Validator
  • Take a screenshot of your page in the browser
  • Save both files in your module1/homework folder

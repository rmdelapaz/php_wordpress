🚀 Theme Performance Optimization

Make your WordPress theme lightning fast

Master techniques for optimal speed and user experience

Learning Objectives

  • Understand Core Web Vitals and performance metrics
  • Optimize images and media files
  • Implement lazy loading effectively
  • Minimize and combine CSS/JavaScript
  • Optimize database queries
  • Implement caching strategies
  • Use CDNs and external resources
  • Optimize for mobile performance

Why Performance Matters

Page speed directly impacts user experience, SEO rankings, and conversion rates. A 1-second delay in page load time can result in a 7% reduction in conversions.

Key Performance Metrics

LCP (Largest Contentful Paint)

< 2.5s
Good target

FID (First Input Delay)

< 100ms
Good target

CLS (Cumulative Layout Shift)

< 0.1
Good target

TTFB (Time to First Byte)

< 600ms
Good target
💡
Core Web Vitals
Google uses Core Web Vitals (LCP, FID, CLS) as ranking factors. Optimizing these metrics improves both user experience and SEO.

Asset Optimization

Optimized Script Loading

<?php
/**
 * Optimize script loading
 */
function mytheme_optimize_scripts() {
    // Remove jQuery migrate
    add_action( 'wp_default_scripts', function( $scripts ) {
        if ( ! is_admin() && isset( $scripts->registered['jquery'] ) ) {
            $script = $scripts->registered['jquery'];
            if ( $script->deps ) {
                $script->deps = array_diff( $script->deps, array( 'jquery-migrate' ) );
            }
        }
    } );
    
    // Move scripts to footer
    remove_action( 'wp_head', 'wp_print_scripts' );
    remove_action( 'wp_head', 'wp_print_head_scripts', 9 );
    add_action( 'wp_footer', 'wp_print_scripts', 5 );
    add_action( 'wp_footer', 'wp_print_head_scripts', 5 );
}
add_action( 'init', 'mytheme_optimize_scripts' );

/**
 * Add async/defer attributes
 */
function mytheme_add_async_defer( $tag, $handle ) {
    // Scripts to load async
    $async_scripts = array( 'mytheme-analytics', 'mytheme-tracking' );
    
    // Scripts to defer
    $defer_scripts = array( 'mytheme-main', 'mytheme-navigation' );
    
    if ( in_array( $handle, $async_scripts ) ) {
        return str_replace( ' src', ' async src', $tag );
    }
    
    if ( in_array( $handle, $defer_scripts ) ) {
        return str_replace( ' src', ' defer src', $tag );
    }
    
    return $tag;
}
add_filter( 'script_loader_tag', 'mytheme_add_async_defer', 10, 2 );

/**
 * Preload critical assets
 */
function mytheme_preload_assets() {
    // Preload fonts
    echo '<link rel="preload" href="' . get_template_directory_uri() . '/fonts/main.woff2" as="font" type="font/woff2" crossorigin>';
    
    // Preload critical CSS
    echo '<link rel="preload" href="' . get_template_directory_uri() . '/css/critical.css" as="style">';
    
    // DNS prefetch for external resources
    echo '<link rel="dns-prefetch" href="//fonts.googleapis.com">';
    echo '<link rel="dns-prefetch" href="//cdn.jsdelivr.net">';
}
add_action( 'wp_head', 'mytheme_preload_assets', 1 );

CSS Optimization

<?php
/**
 * Inline critical CSS
 */
function mytheme_inline_critical_css() {
    $critical_css = file_get_contents( get_template_directory() . '/css/critical.min.css' );
    echo '<style id="critical-css">' . $critical_css . '</style>';
}
add_action( 'wp_head', 'mytheme_inline_critical_css', 5 );

/**
 * Load non-critical CSS asynchronously
 */
function mytheme_async_styles( $html, $handle ) {
    $async_styles = array( 'mytheme-animations', 'mytheme-widgets' );
    
    if ( in_array( $handle, $async_styles ) ) {
        $html = str_replace( "rel='stylesheet'", "rel='preload' as='style' onload=\"this.onload=null;this.rel='stylesheet'\"", $html );
        $html .= '<noscript>' . str_replace( "rel='preload' as='style' onload=\"this.onload=null;this.rel='stylesheet'\"", "rel='stylesheet'", $html ) . '</noscript>';
    }
    
    return $html;
}
add_filter( 'style_loader_tag', 'mytheme_async_styles', 10, 2 );

Image Optimization

Native Lazy Loading

<?php
/**
 * Add native lazy loading
 */
function mytheme_lazy_load_images( $content ) {
    // Skip if in admin or feed
    if ( is_admin() || is_feed() ) {
        return $content;
    }
    
    // Add loading="lazy" to images
    $content = preg_replace( '/<img(.*?)src=/i', '<img$1loading="lazy" src=', $content );
    
    // Add loading="lazy" to iframes
    $content = preg_replace( '/<iframe(.*?)src=/i', '<iframe$1loading="lazy" src=', $content );
    
    return $content;
}
add_filter( 'the_content', 'mytheme_lazy_load_images' );

/**
 * Lazy load with Intersection Observer
 */
function mytheme_advanced_lazy_load() {
    wp_enqueue_script( 'mytheme-lazy', get_template_directory_uri() . '/js/lazy-load.js', array(), '1.0.0', true );
}
add_action( 'wp_enqueue_scripts', 'mytheme_advanced_lazy_load' );

// lazy-load.js
document.addEventListener('DOMContentLoaded', function() {
    const images = document.querySelectorAll('img[data-src]');
    
    const imageObserver = new IntersectionObserver((entries, observer) => {
        entries.forEach(entry => {
            if (entry.isIntersecting) {
                const img = entry.target;
                img.src = img.dataset.src;
                img.removeAttribute('data-src');
                observer.unobserve(img);
            }
        });
    }, {
        rootMargin: '50px 0px',
        threshold: 0.01
    });
    
    images.forEach(img => imageObserver.observe(img));
});

Responsive Images Implementation

<?php
/**
 * Add custom image sizes
 */
function mytheme_custom_image_sizes() {
    add_image_size( 'hero-mobile', 480, 320, true );
    add_image_size( 'hero-tablet', 768, 512, true );
    add_image_size( 'hero-desktop', 1920, 800, true );
    add_image_size( 'card-thumbnail', 400, 300, true );
}
add_action( 'after_setup_theme', 'mytheme_custom_image_sizes' );

/**
 * Generate responsive image markup
 */
function mytheme_responsive_image( $attachment_id, $size = 'full', $class = '' ) {
    $image = wp_get_attachment_image_src( $attachment_id, $size );
    $srcset = wp_get_attachment_image_srcset( $attachment_id, $size );
    $sizes = wp_get_attachment_image_sizes( $attachment_id, $size );
    $alt = get_post_meta( $attachment_id, '_wp_attachment_image_alt', true );
    
    if ( ! $image ) {
        return '';
    }
    
    $html = sprintf(
        '<img src="%s" srcset="%s" sizes="%s" alt="%s" class="%s" loading="lazy" decoding="async">',
        esc_url( $image[0] ),
        esc_attr( $srcset ),
        esc_attr( $sizes ),
        esc_attr( $alt ),
        esc_attr( $class )
    );
    
    return $html;
}

/**
 * Optimize default WordPress srcset
 */
function mytheme_optimize_srcset( $sources ) {
    // Remove unnecessary large sizes
    foreach ( $sources as $key => $source ) {
        if ( $source['descriptor'] == 'w' && $source['value'] > 2000 ) {
            unset( $sources[$key] );
        }
    }
    return $sources;
}
add_filter( 'wp_calculate_image_srcset', 'mytheme_optimize_srcset' );

WebP Image Support

<?php
/**
 * Enable WebP uploads
 */
function mytheme_enable_webp( $mimes ) {
    $mimes['webp'] = 'image/webp';
    return $mimes;
}
add_filter( 'upload_mimes', 'mytheme_enable_webp' );

/**
 * Display WebP in media library
 */
function mytheme_webp_display( $result, $path ) {
    if ( $result === false ) {
        $extension = pathinfo( $path, PATHINFO_EXTENSION );
        if ( $extension === 'webp' ) {
            $result = 'image/webp';
        }
    }
    return $result;
}
add_filter( 'file_is_displayable_image', 'mytheme_webp_display', 10, 2 );

/**
 * Picture element with WebP fallback
 */
function mytheme_picture_element( $attachment_id, $size = 'full' ) {
    $image_url = wp_get_attachment_image_url( $attachment_id, $size );
    $webp_url = str_replace( array( '.jpg', '.jpeg', '.png' ), '.webp', $image_url );
    $alt = get_post_meta( $attachment_id, '_wp_attachment_image_alt', true );
    
    ob_start();
    ?>
    <picture>
        <source srcset="<?php echo esc_url( $webp_url ); ?>" type="image/webp">
        <source srcset="<?php echo esc_url( $image_url ); ?>" type="image/jpeg">
        <img src="<?php echo esc_url( $image_url ); ?>" alt="<?php echo esc_attr( $alt ); ?>" loading="lazy">
    </picture>
    <?php
    return ob_get_clean();
}

Database & Query Optimization

Optimize Database Queries

<?php
/**
 * Cache expensive queries
 */
function mytheme_get_popular_posts( $count = 5 ) {
    $cache_key = 'mytheme_popular_posts_' . $count;
    $posts = get_transient( $cache_key );
    
    if ( false === $posts ) {
        $args = array(
            'posts_per_page'      => $count,
            'meta_key'            => 'post_views',
            'orderby'             => 'meta_value_num',
            'order'               => 'DESC',
            'ignore_sticky_posts' => true,
            'no_found_rows'       => true, // Optimize pagination
            'fields'              => 'ids', // Get only IDs
        );
        
        $query = new WP_Query( $args );
        $posts = $query->posts;
        
        // Cache for 12 hours
        set_transient( $cache_key, $posts, 12 * HOUR_IN_SECONDS );
    }
    
    return $posts;
}

/**
 * Optimize WP_Query
 */
function mytheme_optimized_query( $args = array() ) {
    $defaults = array(
        'no_found_rows'          => true,  // Skip pagination
        'update_post_meta_cache' => false, // Skip meta cache
        'update_post_term_cache' => false, // Skip term cache
    );
    
    $args = wp_parse_args( $args, $defaults );
    return new WP_Query( $args );
}

/**
 * Batch database operations
 */
function mytheme_batch_update_meta() {
    global $wpdb;
    
    // Instead of multiple update_post_meta calls
    $values = array();
    $post_ids = array( 1, 2, 3, 4, 5 );
    
    foreach ( $post_ids as $post_id ) {
        $values[] = $wpdb->prepare( "(%d, 'meta_key', 'meta_value')", $post_id );
    }
    
    if ( ! empty( $values ) ) {
        $query = "INSERT INTO {$wpdb->postmeta} (post_id, meta_key, meta_value) VALUES " . 
                 implode( ',', $values ) . 
                 " ON DUPLICATE KEY UPDATE meta_value = VALUES(meta_value)";
        
        $wpdb->query( $query );
    }
}

Caching Implementation

Object Caching

<?php
/**
 * Fragment caching
 */
function mytheme_cache_fragment( $key, $ttl, $function ) {
    $output = get_transient( $key );
    
    if ( false === $output ) {
        ob_start();
        call_user_func( $function );
        $output = ob_get_clean();
        set_transient( $key, $output, $ttl );
    }
    
    echo $output;
}

// Usage
mytheme_cache_fragment( 'header_menu', HOUR_IN_SECONDS, function() {
    wp_nav_menu( array( 'theme_location' => 'primary' ) );
});

/**
 * Cache API responses
 */
function mytheme_cached_api_call( $url ) {
    $cache_key = 'api_' . md5( $url );
    $data = get_transient( $cache_key );
    
    if ( false === $data ) {
        $response = wp_remote_get( $url );
        
        if ( ! is_wp_error( $response ) ) {
            $data = wp_remote_retrieve_body( $response );
            set_transient( $cache_key, $data, HOUR_IN_SECONDS );
        }
    }
    
    return $data;
}

/**
 * Browser caching headers
 */
function mytheme_browser_cache_headers() {
    if ( ! is_admin() ) {
        header( 'Cache-Control: max-age=31536000, public' );
        header( 'Expires: ' . gmdate( 'D, d M Y H:i:s', time() + 31536000 ) . ' GMT' );
    }
}
add_action( 'send_headers', 'mytheme_browser_cache_headers' );

Mobile Performance

Mobile-Specific Optimizations

<?php
/**
 * Conditional loading for mobile
 */
function mytheme_mobile_optimizations() {
    if ( wp_is_mobile() ) {
        // Dequeue heavy scripts on mobile
        wp_dequeue_script( 'mytheme-animations' );
        wp_dequeue_style( 'mytheme-animations-css' );
        
        // Load mobile-specific styles
        wp_enqueue_style( 
            'mytheme-mobile', 
            get_template_directory_uri() . '/css/mobile.min.css',
            array(),
            '1.0.0'
        );
    }
}
add_action( 'wp_enqueue_scripts', 'mytheme_mobile_optimizations', 20 );

/**
 * Reduce image quality on mobile
 */
function mytheme_mobile_image_quality( $quality, $mime_type ) {
    if ( wp_is_mobile() ) {
        return 70; // Reduce quality to 70%
    }
    return $quality;
}
add_filter( 'wp_editor_set_quality', 'mytheme_mobile_image_quality', 10, 2 );

/**
 * Adaptive content loading
 */
function mytheme_adaptive_content() {
    if ( wp_is_mobile() ) {
        // Show fewer posts on mobile
        add_filter( 'pre_get_posts', function( $query ) {
            if ( ! is_admin() && $query->is_main_query() ) {
                $query->set( 'posts_per_page', 5 );
            }
        });
    }
}

Performance Optimization Checklist

Performance Testing Tools

  • Google PageSpeed Insights: Core Web Vitals and recommendations
  • GTmetrix: Detailed performance analysis
  • WebPageTest: Advanced testing with various locations
  • Chrome DevTools: Lighthouse audits and network analysis
  • Query Monitor: WordPress-specific performance debugging

Performance Best Practices

Development Best Practices

  • Mobile-first approach: Design for mobile performance first
  • Progressive enhancement: Start with core functionality, add features
  • Measure regularly: Test performance throughout development
  • Set performance budgets: Define size and load time limits
  • Optimize critical path: Prioritize above-the-fold content
  • Reduce HTTP requests: Combine files where appropriate
  • Use modern formats: WebP, AVIF for images; WOFF2 for fonts
  • Eliminate render-blocking: Defer non-critical resources
Over-optimization can harm maintainability. Balance performance gains with code complexity and development time.

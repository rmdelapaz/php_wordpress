Course Overview

Welcome to PHP WordPress Development! This comprehensive course will take you from the basics of web development to building professional WordPress websites and applications. Each module consists of multiple 3-hour sessions, delivered one per day.

Course Modules

Duration: 2 weeks (10 sessions)

Get started with the core technologies of web development: HTML, CSS, JavaScript, an introduction to PHP, and development environment setup including Docker basics.

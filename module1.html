Module Overview

This module introduces the core technologies that form the foundation of web development. You'll learn HTML for structure, CSS for styling, JavaScript for interactivity, and get your first introduction to PHP. By the end of this module, you'll be able to build static websites and understand the basics of dynamic web content.

Duration: 2 weeks (10 sessions of 3 hours each)

Prerequisites: None - this is where we start!

Week 1: HTML, CSS, and Development Environment

Session 1: Introduction to Web Development & Setup

  • Course introduction and expectations
  • How the web works: servers, browsers, HTTP
  • Setting up your development environment
    • Installing a code editor (VS Code)
    • Understanding file structures
    • Browser developer tools
    • Introduction to containerization with Docker
    • Installing Docker Desktop
  • Creating your first HTML page
  • Homework: Set up personal GitHub account, create first repository, and install Docker

Session 2: HTML Fundamentals

  • HTML document structure and syntax
  • Essential HTML tags
    • Headings, paragraphs, links
    • Lists, tables, images
    • Semantic HTML elements
  • HTML forms and inputs
  • HTML validation and best practices
  • Homework: Create a personal profile page with HTML

Session 3: CSS Basics

  • Introduction to CSS and its purpose
  • CSS syntax and selectors
  • Inline, internal, and external CSS
  • Working with colors, fonts, and text properties
  • The CSS box model
  • Homework: Add styling to your personal profile page

Session 4: CSS Layout & Responsive Design

  • CSS layout techniques
    • Positioning (relative, absolute, fixed)
    • Float-based layouts
    • Flexbox
    • CSS Grid basics
  • Introduction to responsive design
  • Media queries
  • Mobile-first approach
  • Homework: Make your profile page responsive

Session 5: CSS Frameworks & Best Practices

  • Introduction to Bootstrap
  • Using Bootstrap grid system
  • Bootstrap components and utilities
  • CSS organization and best practices
  • CSS preprocessors overview (SASS/SCSS)
  • Homework: Recreate your profile page using Bootstrap

Week 2: JavaScript and Introduction to PHP

Session 6: JavaScript Fundamentals

  • Introduction to JavaScript and its role in web development
  • JavaScript syntax, variables, and data types
  • Operators and expressions
  • Control flow (conditionals, loops)
  • Functions and scope
  • Homework: Create simple JavaScript programs

Session 7: DOM Manipulation with JavaScript

  • Understanding the Document Object Model (DOM)
  • Selecting and manipulating DOM elements
  • Event handling
  • Creating and removing elements
  • Form validation with JavaScript
  • Homework: Add interactivity to your profile page

Session 8: Modern JavaScript & jQuery

  • ES6+ features overview
  • Introduction to jQuery
  • DOM manipulation with jQuery
  • jQuery animations and effects
  • AJAX basics with jQuery
  • Homework: Refactor your JavaScript code using jQuery

Session 9: Introduction to PHP

  • What is PHP and why it's important for WordPress
  • Setting up a local server environment (XAMPP/MAMP)
  • PHP syntax and basic constructs
  • Variables, data types, and operators
  • Including files and PHP in HTML
  • Homework: Create a simple PHP script

Session 10: Mini-Project: Static Website

  • Planning a multi-page website
  • Creating a consistent layout with HTML and CSS
  • Adding interactivity with JavaScript
  • Basic PHP includes for header and footer
  • Project work time with instructor support
  • Final Project: Complete a 5-page static website with navigation, forms, and styling

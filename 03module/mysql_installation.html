üöÄ Choose Your Operating System

1Download MySQL Installer

Visit the official MySQL website and download the installer:

  1. Go to https://dev.mysql.com/downloads/installer/
  2. Choose "Windows (x86, 32-bit), MSI Installer"
  3. Select the larger file (around 400MB) for the complete package
  4. Click "No thanks, just start my download"

2Run the Installer

Double-click the downloaded MSI file and follow these steps:

3Configuration Settings

  • Port: 3306 (default)
  • Windows Service: MySQL80
  • Start at System Startup: Yes
  • Root Password: Choose a strong one!

1Install Using Homebrew

The easiest way to install MySQL on macOS is using Homebrew:

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

$ brew install mysql

$ brew services start mysql

Alternative: Download the DMG installer from MySQL website

üîß Common Installation Issues

üî¥ MySQL Service Won't Start

Symptoms: Error messages about port 3306 being in use

Solution:

# Check if another MySQL is running
netstat -an | findstr 3306

# Stop conflicting service
net stop mysql

# Restart your MySQL
net start mysql80

üî¥ Forgot Root Password

Don't panic! Here's how to reset it:

  1. Stop MySQL service
  2. Start MySQL with --skip-grant-tables
  3. Connect without password
  4. Reset the password
  5. Restart normally

üî¥ Access Denied Errors

Common causes:

  • Wrong password (check caps lock!)
  • Wrong username (it's 'root' not 'admin')
  • Connection from wrong host

Module Overview

This module focuses on database development with MySQL, which is the database system that powers WordPress. You'll learn database design principles, SQL queries, and how to integrate MySQL with PHP. By the end of this module, you'll be able to design and implement databases for web applications and understand how data is managed in WordPress.

Duration: 1 week (5 sessions of 3 hours each)

Prerequisites: Module 1 & 2: Web Development Fundamentals and PHP Fundamentals

Week 1: MySQL and Database Development

Session 1: Introduction to Databases

  • Database concepts and terminology
  • Types of databases (relational vs. non-relational)
  • Introduction to MySQL
  • MySQL in XAMPP/MAMP environment
  • Using phpMyAdmin
  • Creating databases and tables
  • Homework: Create a database schema for a simple application

Session 2: Database Design

  • Data modeling concepts
  • Entity-Relationship diagrams
  • Normalization principles (1NF, 2NF, 3NF)
  • Primary keys and foreign keys
  • Indexes and their importance
  • Database constraints
  • Homework: Design a normalized database for an e-commerce website

Session 3: SQL Fundamentals

  • SQL syntax and conventions
  • Data types in MySQL
  • Creating and modifying database structures (CREATE, ALTER, DROP)
  • Basic data manipulation (INSERT, UPDATE, DELETE)
  • Simple queries with SELECT
  • Filtering data with WHERE
  • Sorting results with ORDER BY
  • Homework: Write SQL queries to manipulate data in your database

Session 4: Advanced SQL

  • Joins (INNER, LEFT, RIGHT, FULL)
  • Aggregating data (COUNT, SUM, AVG, MIN, MAX)
  • Grouping data with GROUP BY
  • Filtering groups with HAVING
  • Subqueries
  • Views
  • Transactions
  • Stored procedures and functions
  • Homework: Write complex SQL queries with joins and aggregations

Session 5: PHP and MySQL Integration

  • Connecting to MySQL with PHP
  • PDO vs. MySQLi
  • Executing SQL queries through PHP
  • Prepared statements and security concerns
  • CRUD operations in PHP applications
  • Handling results sets
  • Error handling and debugging
  • Final Project: Build a complete data-driven PHP application with MySQL backend

Learning Resources

Recommended Reading

  • SQL Antipatterns by Bill Karwin
  • Online resources: MySQL Documentation, W3Schools SQL Tutorial

Assessment

  • Daily homework assignments: 40%
  • Participation and in-class exercises: 20%
  • Final project (data-driven application): 40%

Important Notes

Understanding database development is crucial for WordPress development, as WordPress utilizes MySQL extensively. The concepts learned in this module will be directly applicable when working with the WordPress database structure in later modules.

This module forms a bridge between the PHP programming knowledge from Module 2 and the WordPress-specific content that follows in Module 4.

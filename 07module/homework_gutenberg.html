Homework: Build a Complete Block Plugin

Create a professional Gutenberg block collection with advanced features

Difficulty: Advanced Estimated Time: 10-12 hours

Project Overview

You will build a comprehensive block plugin called "Content Blocks Pro" that includes multiple custom blocks, patterns, dynamic content, and advanced features. This plugin will demonstrate mastery of Gutenberg development.

Learning Goals

  • Create multiple interconnected blocks
  • Implement dynamic server-side rendering
  • Build reusable block patterns
  • Extend core blocks with filters
  • Create custom format types
  • Implement advanced block controls

Project Requirements

Part 1: Core Blocks Collection

1. Hero Section Block

  • Background image/video/gradient options
  • Overlay with opacity control
  • Heading, subheading, and button components
  • Parallax effect option
  • Responsive height settings
  • Animation on scroll

2. Advanced Testimonial Block

  • Multiple layout options (grid, carousel, masonry)
  • Star rating system
  • Author image and details
  • Company logo support
  • Video testimonials option
  • Schema markup for SEO

3. Pricing Table Block

  • Dynamic number of columns
  • Featured plan highlighting
  • Custom icons for features
  • Toggle for monthly/yearly pricing
  • Currency selector
  • Comparison table mode

4. Team Members Block

  • Grid and carousel layouts
  • Social media links
  • Hover effects with bio
  • Filter by department/role
  • vCard download option
  • Integration with user profiles

5. Advanced Accordion/FAQ Block

  • Nested accordion support
  • Icon customization
  • Schema markup for FAQ
  • Search/filter functionality

Part 3: Block Patterns Library

  • Create 10+ pre-designed patterns
  • Landing page sections
  • Content layouts

Part 4: Block Extensions

  • Add animation controls to all blocks

  • Create custom format toolbar buttons

Dynamic Posts Grid PHP


class Dynamic_Posts_Grid {
    
    public function __construct() {
        add_action('init', [$this, 'register_block']);
        add_action('wp_ajax_load_more_posts', [$this, 'load_more_posts']);
        add_action('wp_ajax_nopriv_load_more_posts', [$this, 'load_more_posts']);
    }
    
    public function register_block() {
        register_block_type('content-blocks-pro/posts-grid', [
            'render_callback' => [$this, 'render'],
            'attributes' => [
                'postsPerPage' => ['type' => 'number', 'default' => 6],
                'categories' => ['type' => 'array', 'default' => []],
                'layout' => ['type' => 'string', 'default' => 'grid'],
                'columns' => ['type' => 'number', 'default' => 3],
                'showExcerpt' => ['type' => 'boolean', 'default' => true],
                'showAuthor' => ['type' => 'boolean', 'default' => false],
                'showDate' => ['type' => 'boolean', 'default' => true],
                'showLoadMore' => ['type' => 'boolean', 'default' => true]
            ]
        ]);
    }
    
    public function render($attributes) {
        $query_args = [
            'posts_per_page' => $attributes['postsPerPage'],
            'post_status' => 'publish',
            'paged' => 1
        ];
        
        if (!empty($attributes['categories'])) {
            $query_args['category__in'] = $attributes['categories'];
        }
        
        $query = new WP_Query($query_args);
        
        ob_start();
        $this->render_posts($query, $attributes);
        return ob_get_clean();
    }
    
    private function render_posts($query, $attributes) {
        if (!$query->have_posts()) {
            echo '
' . __('No posts found', 'content-blocks-pro') . '
';
            return;
        }
        
        $classes = [
            'cbp-posts-grid',
            'layout-' . $attributes['layout'],
            'columns-' . $attributes['columns']
        ];
        
        echo '
';
        
        while ($query->have_posts()) {
            $query->the_post();
            $this->render_post_item($attributes);
        }
        
        echo '
';
        
        if ($attributes['showLoadMore'] && $query->max_num_pages > 1) {
            echo '';
        }
        
        wp_reset_postdata();
    }

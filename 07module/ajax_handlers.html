🔧 Setting up AJAX Handlers

Master server-side AJAX processing in WordPress

Learn to create robust AJAX handlers that process requests and return responses

Learning Objectives

  • Create and register AJAX handler functions
  • Process AJAX requests on the server side
  • Handle different types of data submissions
  • Return appropriate responses (JSON, HTML, etc.)
  • Implement proper security checks
  • Optimize handler performance
  • Debug AJAX handlers effectively

AJAX Handler Structure

Components of an AJAX Handler

1. Hook Registration

Register your handler function with WordPress AJAX hooks

2. Security Verification

Check nonces and user capabilities

3. Data Validation

Validate and sanitize incoming data

4. Business Logic

Process the request and perform operations

5. Response Generation

Create and send the response back to the client

6. Termination

Properly end the handler execution

Basic AJAX Handler Setup

Complete Handler Implementation

 admin_url( 'admin-ajax.php' ),
            'nonce'    => wp_create_nonce( 'my_ajax_nonce' ),
            'loading'  => __( 'Loading...', 'textdomain' ),
            'error'    => __( 'An error occurred', 'textdomain' ),
        ));
    }
    
    /**
     * Handle AJAX request
     */
    public function handle_ajax_request() {
        // 1. Verify nonce
        if ( ! isset( $_POST['nonce'] ) || 
             ! wp_verify_nonce( $_POST['nonce'], 'my_ajax_nonce' ) ) {
            wp_send_json_error( 'Invalid security token' );
        }
        
        // 2. Check user capabilities (if needed)
        if ( ! current_user_can( 'read' ) ) {
            wp_send_json_error( 'Insufficient permissions' );
        }
        
        // 3. Validate and sanitize input
        $data = $this->validate_input( $_POST );
        if ( is_wp_error( $data ) ) {
            wp_send_json_error( $data->get_error_message() );
        }
        
        // 4. Process the request
        $result = $this->process_request( $data );
        
        // 5. Return response
        if ( is_wp_error( $result ) ) {
            wp_send_json_error( array(
                'message' => $result->get_error_message(),
                'code'    => $result->get_error_code(),
            ));
        } else {
            wp_send_json_success( array(
                'message' => 'Request processed successfully',
                'data'    => $result,
            ));
        }
    }
    
    /**
     * Validate input data
     */
    private function validate_input( $post_data ) {
        $validated = array();
        
        // Validate required fields
        if ( empty( $post_data['user_input'] ) ) {
            return new WP_Error( 'missing_input', 'Required field is missing' );
        }
        
        // Sanitize data
        $validated['user_input'] = sanitize_text_field( $post_data['user_input'] );
        
        // Validate specific formats
        if ( isset( $post_data['email'] ) ) {
            $email = sanitize_email( $post_data['email'] );
            if ( ! is_email( $email ) ) {
                return new WP_Error( 'invalid_email', 'Invalid email address' );
            }
            $validated['email'] = $email;
        }
        
        return $validated;
    }
    
    /**
     * Process the actual request
     */
    private function process_request( $data ) {
        // Perform your business logic here
        // Example: Save to database
        global $wpdb;
        
        $result = $wpdb->insert(
            $wpdb->prefix . 'my_table',
            array(
                'user_input' => $data['user_input'],
                'created_at' => current_time( 'mysql' ),
            ),
            array( '%s', '%s' )
        );
        
        if ( false === $result ) {
            return new WP_Error( 'db_error', 'Failed to save data' );
        }
        
        return array(
            'id'      => $wpdb->insert_id,
            'message' => 'Data saved successfully',
        );
    }
}

Contact Form Handler

function handle_contact_form() {
    // Verify nonce
    if ( ! wp_verify_nonce( $_POST['nonce'], 'contact_form_nonce' ) ) {
        wp_send_json_error( 'Security check failed' );
    }
    
    // Validate fields
    $errors = array();
    
    $name = sanitize_text_field( $_POST['name'] ?? '' );
    if ( empty( $name ) ) {
        $errors['name'] = 'Name is required';
    }
    
    $email = sanitize_email( $_POST['email'] ?? '' );
    if ( ! is_email( $email ) ) {
        $errors['email'] = 'Valid email is required';
    }
    
    $message = sanitize_textarea_field( $_POST['message'] ?? '' );
    if ( strlen( $message ) < 10 ) {
        $errors['message'] = 'Message must be at least 10 characters';
    }
    
    // Return errors if any
    if ( ! empty( $errors ) ) {
        wp_send_json_error( array(
            'errors' => $errors,
            'message' => 'Please fix the errors and try again',
        ));
    }
    
    // Process form (send email, save to database, etc.)
    $sent = wp_mail(
        get_option( 'admin_email' ),
        'Contact Form Submission',
        sprintf( "Name: %s\nEmail: %s\nMessage: %s", $name, $email, $message ),
        array( 'From: ' . $email )
    );
    
    if ( $sent ) {
        // Save to database
        global $wpdb;
        $wpdb->insert(
            $wpdb->prefix . 'contact_submissions',
            array(
                'name' => $name,
                'email' => $email,
                'message' => $message,
                'submitted_at' => current_time( 'mysql' ),
            )
        );
        
        wp_send_json_success( array(
            'message' => 'Thank you! Your message has been sent.',
            'redirect' => home_url( '/thank-you/' ),
        ));
    } else {
        wp_send_json_error( 'Failed to send message. Please try again.' );
    }
}
add_action( 'wp_ajax_submit_contact', 'handle_contact_form' );
add_action( 'wp_ajax_nopriv_submit_contact', 'handle_contact_form' );

Live Search Handler

function handle_live_search() {
    // Check nonce
    if ( ! wp_verify_nonce( $_GET['nonce'], 'search_nonce' ) ) {
        wp_send_json_error( 'Invalid nonce' );
    }
    
    // Get and sanitize search term
    $search_term = sanitize_text_field( $_GET['search'] ?? '' );
    
    if ( strlen( $search_term ) < 3 ) {
        wp_send_json_error( 'Search term must be at least 3 characters' );
    }
    
    // Perform search
    $args = array(
        'post_type' => array( 'post', 'page', 'product' ),
        'post_status' => 'publish',
        's' => $search_term,
        'posts_per_page' => 10,
        'orderby' => 'relevance',
    );
    
    $query = new WP_Query( $args );
    $results = array();
    
    if ( $query->have_posts() ) {
        while ( $query->have_posts() ) {
            $query->the_post();
            
            $results[] = array(
                'id' => get_the_ID(),
                'title' => get_the_title(),
                'excerpt' => wp_trim_words( get_the_excerpt(), 15 ),
                'url' => get_permalink(),
                'type' => get_post_type(),
                'thumbnail' => get_the_post_thumbnail_url( get_the_ID(), 'thumbnail' ),
            );
        }
        wp_reset_postdata();
    }
    
    // Return results
    wp_send_json_success( array(
        'results' => $results,
        'count' => count( $results ),
        'search_term' => $search_term,
        'message' => sprintf( 'Found %d results for "%s"', count( $results ), $search_term ),
    ));
}
add_action( 'wp_ajax_live_search', 'handle_live_search' );
add_action( 'wp_ajax_nopriv_live_search', 'handle_live_search' );

File Upload Handler

function handle_file_upload() {
    // Check nonce
    if ( ! wp_verify_nonce( $_POST['nonce'], 'upload_nonce' ) ) {
        wp_send_json_error( 'Security check failed' );
    }
    
    // Check user capabilities
    if ( ! current_user_can( 'upload_files' ) ) {
        wp_send_json_error( 'You do not have permission to upload files' );
    }
    
    // Check if file was uploaded
    if ( empty( $_FILES['file'] ) ) {
        wp_send_json_error( 'No file uploaded' );
    }
    
    $file = $_FILES['file'];
    
    // Validate file type
    $allowed_types = array( 'jpg', 'jpeg', 'png', 'gif', 'pdf' );
    $file_type = wp_check_filetype( $file['name'] );
    
    if ( ! in_array( $file_type['ext'], $allowed_types ) ) {
        wp_send_json_error( 'Invalid file type. Allowed: ' . implode( ', ', $allowed_types ) );
    }
    
    // Validate file size (5MB max)
    $max_size = 5 * 1024 * 1024; // 5MB in bytes
    if ( $file['size'] > $max_size ) {
        wp_send_json_error( 'File size exceeds 5MB limit' );
    }
    
    // Handle upload
    require_once( ABSPATH . 'wp-admin/includes/file.php' );
    require_once( ABSPATH . 'wp-admin/includes/media.php' );
    require_once( ABSPATH . 'wp-admin/includes/image.php' );
    
    $upload = wp_handle_upload( $file, array( 'test_form' => false ) );
    
    if ( isset( $upload['error'] ) ) {
        wp_send_json_error( $upload['error'] );
    }
    
    // Create attachment
    $attachment_id = wp_insert_attachment(
        array(
            'post_mime_type' => $upload['type'],
            'post_title' => sanitize_file_name( $file['name'] ),
            'post_content' => '',
            'post_status' => 'inherit',
        ),
        $upload['file']
    );
    
    // Generate metadata
    wp_update_attachment_metadata(
        $attachment_id,
        wp_generate_attachment_metadata( $attachment_id, $upload['file'] )
    );
    
    // Return success with file info
    wp_send_json_success( array(
        'message' => 'File uploaded successfully',
        'attachment_id' => $attachment_id,
        'url' => $upload['url'],
        'type' => $upload['type'],
        'size' => size_format( $file['size'] ),
    ));
}
add_action( 'wp_ajax_upload_file', 'handle_file_upload' );

Database Operations in Handlers

CRUD Operations Handler

class AJAX_CRUD_Handler {
    
    public function __construct() {
        add_action( 'wp_ajax_crud_create', array( $this, 'create' ) );
        add_action( 'wp_ajax_crud_read', array( $this, 'read' ) );
        add_action( 'wp_ajax_crud_update', array( $this, 'update' ) );
        add_action( 'wp_ajax_crud_delete', array( $this, 'delete' ) );
    }
    
    /**
     * Create record
     */
    public function create() {
        $this->verify_request();
        
        global $wpdb;
        $table = $wpdb->prefix . 'custom_table';
        
        $data = array(
            'title' => sanitize_text_field( $_POST['title'] ),
            'content' => sanitize_textarea_field( $_POST['content'] ),
            'user_id' => get_current_user_id(),
            'created_at' => current_time( 'mysql' ),
        );
        
        $result = $wpdb->insert( $table, $data );
        
        if ( false === $result ) {
            wp_send_json_error( 'Failed to create record' );
        }
        
        wp_send_json_success( array(
            'id' => $wpdb->insert_id,
            'message' => 'Record created successfully',
        ));
    }
    
    /**
     * Read records
     */
    public function read() {
        $this->verify_request();
        
        global $wpdb;
        $table = $wpdb->prefix . 'custom_table';
        
        // Pagination
        $page = isset( $_GET['page'] ) ? absint( $_GET['page'] ) : 1;
        $per_page = 10;
        $offset = ( $page - 1 ) * $per_page;
        
        // Get records
        $records = $wpdb->get_results(
            $wpdb->prepare(
                "SELECT * FROM $table 
                ORDER BY created_at DESC 
                LIMIT %d OFFSET %d",
                $per_page,
                $offset
            )
        );
        
        // Get total count
        $total = $wpdb->get_var( "SELECT COUNT(*) FROM $table" );
        
        wp_send_json_success( array(
            'records' => $records,
            'pagination' => array(
                'page' => $page,
                'per_page' => $per_page,
                'total' => $total,
                'pages' => ceil( $total / $per_page ),
            ),
        ));
    }
    
    /**
     * Update record
     */
    public function update() {
        $this->verify_request();
        
        $id = absint( $_POST['id'] );
        if ( ! $id ) {
            wp_send_json_error( 'Invalid ID' );
        }
        
        global $wpdb;
        $table = $wpdb->prefix . 'custom_table';
        
        $data = array(
            'title' => sanitize_text_field( $_POST['title'] ),
            'content' => sanitize_textarea_field( $_POST['content'] ),
            'updated_at' => current_time( 'mysql' ),
        );
        
        $result = $wpdb->update(
            $table,
            $data,
            array( 'id' => $id ),
            null,
            array( '%d' )
        );
        
        if ( false === $result ) {
            wp_send_json_error( 'Failed to update record' );
        }
        
        wp_send_json_success( array(
            'message' => 'Record updated successfully',
            'updated' => $result,
        ));
    }
    
    /**
     * Delete record
     */
    public function delete() {
        $this->verify_request();
        
        $id = absint( $_POST['id'] );
        if ( ! $id ) {
            wp_send_json_error( 'Invalid ID' );
        }
        
        global $wpdb;
        $table = $wpdb->prefix . 'custom_table';
        
        $result = $wpdb->delete(
            $table,
            array( 'id' => $id ),
            array( '%d' )
        );
        
        if ( false === $result ) {
            wp_send_json_error( 'Failed to delete record' );
        }
        
        wp_send_json_success( array(
            'message' => 'Record deleted successfully',
            'deleted' => $result,
        ));
    }
    
    /**
     * Verify request
     */
    private function verify_request() {
        if ( ! wp_verify_nonce( $_REQUEST['nonce'], 'crud_nonce' ) ) {
            wp_send_json_error( 'Invalid nonce' );
        }
        
        if ( ! current_user_can( 'manage_options' ) ) {
            wp_send_json_error( 'Insufficient permissions' );
        }
    }
}

Single Handler for Multiple Actions

class Multi_Action_Handler {
    
    public function __construct() {
        add_action( 'wp_ajax_multi_handler', array( $this, 'route_action' ) );
    }
    
    /**
     * Route to appropriate method based on action
     */
    public function route_action() {
        // Verify nonce
        if ( ! wp_verify_nonce( $_REQUEST['nonce'], 'multi_nonce' ) ) {
            wp_send_json_error( 'Invalid nonce' );
        }
        
        // Get sub-action
        $sub_action = sanitize_text_field( $_REQUEST['sub_action'] ?? '' );
        
        // Route to appropriate method
        switch ( $sub_action ) {
            case 'get_data':
                $this->get_data();
                break;
                
            case 'save_data':
                $this->save_data();
                break;
                
            case 'delete_data':
                $this->delete_data();
                break;
                
            case 'export_data':
                $this->export_data();
                break;
                
            default:
                wp_send_json_error( 'Invalid action: ' . $sub_action );
        }
    }
    
    private function get_data() {
        // Implementation
        wp_send_json_success( array( 'data' => 'Retrieved data' ) );
    }
    
    private function save_data() {
        // Implementation
        wp_send_json_success( array( 'message' => 'Data saved' ) );
    }
    
    private function delete_data() {
        // Implementation
        wp_send_json_success( array( 'message' => 'Data deleted' ) );
    }
    
    private function export_data() {
        // Implementation
        wp_send_json_success( array( 'file' => 'export.csv' ) );
    }
}

Performance Optimization

Optimized Handler Techniques

class Optimized_AJAX_Handler {
    
    /**
     * Cached handler with transients
     */
    public function cached_handler() {
        check_ajax_referer( 'cached_nonce', 'nonce' );
        
        $cache_key = 'ajax_cache_' . md5( serialize( $_REQUEST ) );
        $cached = get_transient( $cache_key );
        
        if ( false !== $cached ) {
            wp_send_json_success( array(
                'data' => $cached,
                'cached' => true,
            ));
        }
        
        // Expensive operation
        $result = $this->expensive_operation();
        
        // Cache for 1 hour
        set_transient( $cache_key, $result, HOUR_IN_SECONDS );
        
        wp_send_json_success( array(
            'data' => $result,
            'cached' => false,
        ));
    }
    
    /**
     * Batch processing handler
     */
    public function batch_handler() {
        check_ajax_referer( 'batch_nonce', 'nonce' );
        
        $batch_size = 50;
        $offset = absint( $_POST['offset'] ?? 0 );
        
        global $wpdb;
        $items = $wpdb->get_results(
            $wpdb->prepare(
                "SELECT * FROM {$wpdb->posts} 
                WHERE post_type = 'post' 
                LIMIT %d OFFSET %d",
                $batch_size,
                $offset
            )
        );
        
        // Process items
        $processed = array();
        foreach ( $items as $item ) {
            $processed[] = $this->process_item( $item );
        }
        
        // Check if more items exist
        $has_more = count( $items ) === $batch_size;
        
        wp_send_json_success( array(
            'processed' => count( $processed ),
            'offset' => $offset + $batch_size,
            'has_more' => $has_more,
            'message' => sprintf( 'Processed %d items', count( $processed ) ),
        ));
    }
    
    /**
     * Debounced handler
     */
    public function debounced_search() {
        check_ajax_referer( 'search_nonce', 'nonce' );
        
        $search_id = sanitize_text_field( $_POST['search_id'] );
        
        // Store search timestamp
        $transient_key = 'search_' . $search_id;
        $last_search = get_transient( $transient_key );
        
        // If searched within last 500ms, ignore
        if ( $last_search && ( time() - $last_search ) < 0.5 ) {
            wp_send_json_error( 'Too many requests' );
        }
        
        set_transient( $transient_key, time(), 1 );
        
        // Perform search
        $results = $this->perform_search( $_POST['query'] );
        
        wp_send_json_success( $results );
    }
}

/**
 * Debug-friendly AJAX handler
 */
function debug_ajax_handler() {
    // Enable error reporting for debugging
    if ( WP_DEBUG ) {
        error_reporting( E_ALL );
        ini_set( 'display_errors', 1 );
    }
    
    // Log incoming request
    error_log( 'AJAX Request: ' . print_r( $_REQUEST, true ) );
    
    try {
        // Verify nonce
        if ( ! wp_verify_nonce( $_REQUEST['nonce'], 'debug_nonce' ) ) {
            throw new Exception( 'Invalid nonce' );
        }
        
        // Process request
        $result = process_request();
        
        // Log successful result
        error_log( 'AJAX Success: ' . print_r( $result, true ) );
        
        // Add debug info in development
        if ( WP_DEBUG ) {
            $result['debug'] = array(
                'memory_usage' => memory_get_peak_usage( true ),
                'execution_time' => microtime( true ) - $_SERVER['REQUEST_TIME_FLOAT'],
                'queries' => get_num_queries(),
            );
        }
        
        wp_send_json_success( $result );
        
    } catch ( Exception $e ) {
        // Log error
        error_log( 'AJAX Error: ' . $e->getMessage() );
        error_log( 'Stack trace: ' . $e->getTraceAsString() );
        
        // Send error response
        wp_send_json_error( array(
            'message' => $e->getMessage(),
            'code' => $e->getCode(),
            'file' => WP_DEBUG ? $e->getFile() : '',
            'line' => WP_DEBUG ? $e->getLine() : '',
        ));
    }
}

Best Practices

✅ Do's

  • Always validate and sanitize input
  • Use appropriate response functions (wp_send_json_*)
  • Implement proper error handling
  • Keep handlers focused on single responsibilities
  • Use transients for caching when appropriate
  • Document your AJAX endpoints
  • Test with different user roles

❌ Don'ts

  • Don't skip security checks
  • Don't output anything before wp_send_json functions
  • Don't forget to terminate with wp_die()
  • Don't trust client-side validation alone
  • Don't expose sensitive information
  • Don't perform heavy operations without pagination

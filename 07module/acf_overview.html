Advanced Custom Fields Plugin Overview

Master the most powerful custom fields plugin for WordPress

Learning Objectives

By the end of this lesson, you will be able to:

  • Understand ACF architecture and capabilities
  • Install and configure ACF Pro
  • Create and manage field groups
  • Work with different field types
  • Display field data in templates
  • Use ACF for complex content structures

What is Advanced Custom Fields?

Advanced Custom Fields (ACF) is a WordPress plugin that allows you to add custom fields to WordPress edit screens with ease. It's the most popular solution for creating custom content in WordPress.

Why Use ACF?

  • User-Friendly Interface: Intuitive UI for creating fields
  • 30+ Field Types: Text, textarea, select, checkbox, image, file, relationship, and more
  • Flexible Locations: Add fields to posts, pages, users, taxonomies, options pages
  • Developer-Friendly: Simple functions for displaying field data
  • Conditional Logic: Show/hide fields based on other field values
  • Repeater Fields: Create unlimited rows of data
  • Flexible Content: Build page layouts with blocks

Installation and Setup

Installing ACF


# Via WordPress Admin
1. Go to Plugins → Add New
2. Search for "Advanced Custom Fields"
3. Install and Activate

# Via Composer (ACF Pro)
composer require advanced-custom-fields/advanced-custom-fields-pro

# Via WP-CLI
wp plugin install advanced-custom-fields --activate

Basic Configuration


// functions.php - Configure ACF settings
add_filter('acf/settings/url', 'my_acf_settings_url');
function my_acf_settings_url($url) {
    return MY_PLUGIN_URL . '/includes/acf/';
}

add_filter('acf/settings/show_admin', 'my_acf_settings_show_admin');
function my_acf_settings_show_admin($show_admin) {
    // Hide ACF admin for non-admins
    return current_user_can('manage_options');
}

// Save ACF JSON
add_filter('acf/settings/save_json', 'my_acf_json_save_point');
function my_acf_json_save_point($path) {
    $path = get_stylesheet_directory() . '/acf-json';
    return $path;
}

// Load ACF JSON
add_filter('acf/settings/load_json', 'my_acf_json_load_point');
function my_acf_json_load_point($paths) {
    unset($paths[0]);
    $paths[] = get_stylesheet_directory() . '/acf-json';
    return $paths;
}

Creating Field Groups

Via Admin Interface

  1. Navigate to Custom Fields → Field Groups

  3. Enter a title for your field group

  6. Configure settings and publish

Displaying Field Data

Basic Functions


// Get field value
$value = get_field('field_name');
$value = get_field('field_name', $post_id);

// Display field value (with automatic escaping)
the_field('field_name');
the_field('field_name', $post_id);

// Get field object (includes all settings)
$field = get_field_object('field_name');
$field = get_field_object('field_key');

// Check if field has value
if (get_field('field_name')) {
    // Field has value
}

// Get all fields for a post
$fields = get_fields();
$fields = get_fields($post_id);

// Get fields with formatting
$value = get_field('field_name', $post_id, true); // Format value
$value = get_field('field_name', $post_id, false); // Don't format value

Working with Different Field Types


// Image field
$image = get_field('image_field');
if ($image) {
    // When return format is 'array'
    echo '<img src="' . esc_url($image['url']) . '" alt="' . esc_attr($image['alt']) . '">';
    
    // Access different sizes
    echo '<img src="' . esc_url($image['sizes']['thumbnail']) . '">';
    
    // When return format is 'url'
    echo '<img src="' . esc_url($image) . '">';
    
    // When return format is 'id'
    echo wp_get_attachment_image($image, 'full');
}

// Link field
$link = get_field('link_field');
if ($link) {
    $link_url = $link['url'];
    $link_title = $link['title'];
    $link_target = $link['target'] ? $link['target'] : '_self';
    echo '<a href="' . esc_url($link_url) . '" target="' . esc_attr($link_target) . '">' . esc_html($link_title) . '</a>';
}

// Relationship field
$posts = get_field('relationship_field');
if ($posts) {
    echo '<ul>';
    foreach ($posts as $post) {
        setup_postdata($post);
        echo '<li><a href="' . get_permalink() . '">' . get_the_title() . '</a></li>';
    }
    wp_reset_postdata();
    echo '</ul>';
}

// Select field with multiple values
$values = get_field('multi_select_field');
if ($values) {
    echo '<ul>';
    foreach ($values as $value) {
        echo '<li>' . esc_html($value) . '</li>';
    }
    echo '</ul>';
}

ACF Best Practices

Development Workflow

  • Use Local JSON: Enable ACF JSON for version control
  • Register via PHP: Use PHP for production field registration
  • Escape Output: Always escape field values when displaying
  • Check for ACF: Verify ACF is active before using functions
  • Use Field Keys: Reference fields by key for reliability
  • Optimize Queries: Use get_field() wisely in loops

Performance Optimization


// Cache field values in loops
$posts = get_posts(['post_type' => 'product', 'posts_per_page' => -1]);
$prices = [];

foreach ($posts as $post) {
    // Inefficient - multiple database queries
    // $price = get_field('price', $post->ID);
    
    // Efficient - single query per post
    $fields = get_fields($post->ID);
    $prices[$post->ID] = $fields['price'];
}

// Use update_field for better performance
update_field('field_key', $value, $post_id);

// Disable formatting when not needed
$raw_value = get_field('field_name', $post_id, false);

Understanding the Settings API

The WordPress Settings API provides a standardized way to create plugin settings pages. It handles security, data validation, and saving automatically.

Settings API Implementation Flow

Step-by-Step Process

1

Register Settings

Use register_setting() to register your settings and validation callback

2

Add Settings Sections

Core Settings API Functions

register_setting()

Registers a setting and its sanitization callback.

Parameter Type Description
$option_group string Settings group name
$option_name string Option name in database
$args array Optional. Array of arguments

add_settings_section()

Adds a new section to a settings page.

Parameter Type Description
$id string Section ID
$title string Section title
$callback callable Function to output section content
$page string Page slug where section appears

add_settings_field()

Adds a field to a settings section.

Parameter Type Description
$id string Field ID
$title string Field title/label
$callback callable Function to output field HTML
$page string Settings page slug
$section string Section ID
$args array Optional. Additional arguments

Complete Settings API Implementation

Full Working Example

<?php
/**
 * Plugin Settings using WordPress Settings API
 */
class My_Plugin_Settings {
    
    private $options;
    
    public function __construct() {
        add_action( 'admin_menu', array( $this, 'add_settings_page' ) );
        add_action( 'admin_init', array( $this, 'register_settings' ) );
        
        // Get current options
        $this->options = get_option( 'my_plugin_options' );
    }
    
    /**
     * Add settings page to admin menu
     */
    public function add_settings_page() {
        add_options_page(
            'My Plugin Settings',     // Page title
            'My Plugin',              // Menu title
            'manage_options',         // Capability
            'my-plugin-settings',     // Menu slug
            array( $this, 'render_settings_page' )  // Callback
        );
    }
    
    /**
     * Register settings, sections, and fields
     */
    public function register_settings() {
        // Register the settings
        register_setting(
            'my_plugin_settings_group',     // Option group
            'my_plugin_options',            // Option name
            array( $this, 'sanitize_options' )  // Sanitize callback
        );
        
        // Add General Settings section
        add_settings_section(
            'my_plugin_general_section',    // ID
            'General Settings',              // Title
            array( $this, 'render_general_section' ),  // Callback
            'my-plugin-settings'            // Page
        );
        
        // Add API Key field
        add_settings_field(
            'api_key',                      // ID
            'API Key',                      // Title
            array( $this, 'render_api_key_field' ),  // Callback
            'my-plugin-settings',           // Page
            'my_plugin_general_section',    // Section
            array(                           // Args
                'label_for' => 'api_key',
                'description' => 'Enter your API key from the service provider'
            )
        );
        
        // Add Enable Feature checkbox
        add_settings_field(
            'enable_feature',
            'Enable Feature',
            array( $this, 'render_checkbox_field' ),
            'my-plugin-settings',
            'my_plugin_general_section',
            array(
                'label_for' => 'enable_feature',
                'description' => 'Enable the main plugin feature'
            )
        );
        
        // Add Display Settings section
        add_settings_section(
            'my_plugin_display_section',
            'Display Settings',
            array( $this, 'render_display_section' ),
            'my-plugin-settings'
        );
        
        // Add Color Scheme field
        add_settings_field(
            'color_scheme',
            'Color Scheme',
            array( $this, 'render_select_field' ),
            'my-plugin-settings',
            'my_plugin_display_section',
            array(
                'label_for' => 'color_scheme',
                'options' => array(
                    'light' => 'Light',
                    'dark' => 'Dark',
                    'auto' => 'Automatic'
                )
            )
        );
        
        // Add Items Per Page field
        add_settings_field(
            'items_per_page',
            'Items Per Page',
            array( $this, 'render_number_field' ),
            'my-plugin-settings',
            'my_plugin_display_section',
            array(
                'label_for' => 'items_per_page',
                'min' => 1,
                'max' => 100,
                'default' => 10
            )
        );
    }
    
    /**
     * Render the settings page
     */
    public function render_settings_page() {
        ?>
        <div class="wrap">
            <h1><?php echo esc_html( get_admin_page_title() ); ?></h1>
            
            <?php settings_errors(); ?>
            
            <form method="post" action="options.php">
                <?php
                // Output security fields
                settings_fields( 'my_plugin_settings_group' );
                
                // Output settings sections and fields
                do_settings_sections( 'my-plugin-settings' );
                
                // Submit button
                submit_button( 'Save Settings' );
                ?>
            </form>
        </div>
        <?php
    }
    
    /**
     * Render General section description
     */
    public function render_general_section() {
        echo '<p>Configure the general settings for the plugin.</p>';
    }
    
    /**
     * Render Display section description
     */
    public function render_display_section() {
        echo '<p>Customize how content is displayed.</p>';
    }
    
    /**
     * Render API Key field
     */
    public function render_api_key_field( $args ) {
        $value = isset( $this->options['api_key'] ) ? $this->options['api_key'] : '';
        ?>
        <input type="text" 
               id="<?php echo esc_attr( $args['label_for'] ); ?>" 
               name="my_plugin_options[api_key]" 
               value="<?php echo esc_attr( $value ); ?>" 
               class="regular-text" />
        <p class="description">
            <?php echo esc_html( $args['description'] ); ?>
        </p>
        <?php
    }
    
    /**
     * Render checkbox field
     */
    public function render_checkbox_field( $args ) {
        $value = isset( $this->options['enable_feature'] ) ? $this->options['enable_feature'] : 0;
        ?>
        <input type="checkbox" 
               id="<?php echo esc_attr( $args['label_for'] ); ?>" 
               name="my_plugin_options[enable_feature]" 
               value="1" 
               <?php checked( $value, 1 ); ?> />
        <label for="<?php echo esc_attr( $args['label_for'] ); ?>">
            <?php echo esc_html( $args['description'] ); ?>
        </label>
        <?php
    }
    
    /**
     * Render select field
     */
    public function render_select_field( $args ) {
        $value = isset( $this->options['color_scheme'] ) ? $this->options['color_scheme'] : 'light';
        ?>
        <select id="<?php echo esc_attr( $args['label_for'] ); ?>" 
                name="my_plugin_options[color_scheme]">
            <?php foreach ( $args['options'] as $key => $label ) : ?>
                <option value="<?php echo esc_attr( $key ); ?>" 
                        <?php selected( $value, $key ); ?>>
                    <?php echo esc_html( $label ); ?>
                </option>
            <?php endforeach; ?>
        </select>
        <?php
    }
    
    /**
     * Render number field
     */
    public function render_number_field( $args ) {
        $value = isset( $this->options['items_per_page'] ) 
            ? $this->options['items_per_page'] 
            : $args['default'];
        ?>
        <input type="number" 
               id="<?php echo esc_attr( $args['label_for'] ); ?>" 
               name="my_plugin_options[items_per_page]" 
               value="<?php echo esc_attr( $value ); ?>" 
               min="<?php echo esc_attr( $args['min'] ); ?>" 
               max="<?php echo esc_attr( $args['max'] ); ?>" 
               class="small-text" />
        <?php
    }
    
    /**
     * Sanitize and validate options
     */
    public function sanitize_options( $input ) {
        $sanitized = array();
        
        // Sanitize API Key
        if ( isset( $input['api_key'] ) ) {
            $sanitized['api_key'] = sanitize_text_field( $input['api_key'] );
            
            // Validate API key format (example)
            if ( ! preg_match( '/^[A-Z0-9]{32}$/i', $sanitized['api_key'] ) ) {
                add_settings_error(
                    'api_key',
                    'invalid_api_key',
                    'API Key must be 32 alphanumeric characters',
                    'error'
                );
                $sanitized['api_key'] = '';
            }
        }
        
        // Sanitize checkbox
        $sanitized['enable_feature'] = isset( $input['enable_feature'] ) ? 1 : 0;
        
        // Sanitize select
        if ( isset( $input['color_scheme'] ) ) {
            $valid_schemes = array( 'light', 'dark', 'auto' );
            if ( in_array( $input['color_scheme'], $valid_schemes ) ) {
                $sanitized['color_scheme'] = $input['color_scheme'];
            } else {
                $sanitized['color_scheme'] = 'light';
            }
        }
        
        // Sanitize number
        if ( isset( $input['items_per_page'] ) ) {
            $sanitized['items_per_page'] = absint( $input['items_per_page'] );
            
            // Validate range
            if ( $sanitized['items_per_page'] < 1 || $sanitized['items_per_page'] > 100 ) {
                add_settings_error(
                    'items_per_page',
                    'invalid_range',
                    'Items per page must be between 1 and 100',
                    'error'
                );
                $sanitized['items_per_page'] = 10;
            }
        }
        
        return $sanitized;
    }
}

// Initialize the settings
new My_Plugin_Settings();

Custom Field Types

<?php
// Color picker field
public function render_color_picker_field( $args ) {
    $value = isset( $this->options['theme_color'] ) 
        ? $this->options['theme_color'] 
        : '#0073aa';
    ?>
    <input type="text" 
           id="<?php echo esc_attr( $args['label_for'] ); ?>" 
           name="my_plugin_options[theme_color]" 
           value="<?php echo esc_attr( $value ); ?>" 
           class="color-field" 
           data-default-color="#0073aa" />
    <?php
}

// Media upload field
public function render_media_upload_field( $args ) {
    $value = isset( $this->options['logo_id'] ) ? $this->options['logo_id'] : '';
    $image_url = $value ? wp_get_attachment_url( $value ) : '';
    ?>
    <div class="media-upload-wrapper">
        <input type="hidden" 
               id="<?php echo esc_attr( $args['label_for'] ); ?>" 
               name="my_plugin_options[logo_id]" 
               value="<?php echo esc_attr( $value ); ?>" />
        
        <div id="preview-image">
            <?php if ( $image_url ) : ?>
                <img src="<?php echo esc_url( $image_url ); ?>" style="max-width: 200px;" />
            <?php endif; ?>
        </div>
        
        <button type="button" class="button upload-button">Select Image</button>
        <button type="button" class="button remove-button">Remove</button>
    </div>
    <?php
}

// Multi-checkbox field
public function render_multi_checkbox_field( $args ) {
    $values = isset( $this->options['features'] ) 
        ? $this->options['features'] 
        : array();
    
    foreach ( $args['options'] as $key => $label ) {
        $checked = in_array( $key, $values );
        ?>
        <label style="display: block; margin-bottom: 5px;">
            <input type="checkbox" 
                   name="my_plugin_options[features][]" 
                   value="<?php echo esc_attr( $key ); ?>" 
                   <?php checked( $checked ); ?> />
            <?php echo esc_html( $label ); ?>
        </label>
        <?php
    }
}

Advanced Validation

<?php
public function sanitize_options( $input ) {
    $sanitized = array();
    $errors = false;
    
    // Validate email
    if ( isset( $input['admin_email'] ) ) {
        $email = sanitize_email( $input['admin_email'] );
        if ( ! is_email( $email ) ) {
            add_settings_error(
                'admin_email',
                'invalid_email',
                'Please enter a valid email address',
                'error'
            );
            $errors = true;
        } else {
            $sanitized['admin_email'] = $email;
        }
    }
    
    // Validate URL
    if ( isset( $input['webhook_url'] ) ) {
        $url = esc_url_raw( $input['webhook_url'] );
        if ( ! filter_var( $url, FILTER_VALIDATE_URL ) ) {
            add_settings_error(
                'webhook_url',
                'invalid_url',
                'Please enter a valid URL',
                'error'
            );
            $errors = true;
        } else {
            $sanitized['webhook_url'] = $url;
        }
    }
    
    // If errors, return old values for error fields
    if ( $errors ) {
        $old_options = get_option( 'my_plugin_options' );
        return array_merge( $old_options, $sanitized );
    }
    
    return $sanitized;
}

Settings API Best Practices

  • Group related settings: Use sections to organize settings logically
  • Validate everything: Never trust user input, always validate and sanitize
  • Provide defaults: Have sensible defaults for all settings
  • Use descriptive IDs: Make field IDs meaningful and unique
  • Add help text: Provide descriptions for complex settings
  • Handle errors gracefully: Show clear error messages
  • Store as single option: Group settings in one database option
  • Escape output: Always escape when displaying values
Always use nonces (handled automatically by settings_fields()) to protect against CSRF attacks!
Store all plugin settings as a single array option to minimize database queries.

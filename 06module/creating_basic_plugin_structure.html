Your First WordPress Plugin

Creating a WordPress plugin starts with understanding the basic structure and required components. Let's build a simple but complete plugin step by step.

What We'll Build

We'll create a "Hello World Plus" plugin that:

  • Displays a customizable message
  • Has an admin settings page
  • Includes a shortcode
  • Adds a widget
  • Uses proper WordPress standards

Step-by-Step Plugin Creation

1

Create Plugin Folder

Navigate to your WordPress plugins directory and create a new folder:

cd wp-content/plugins
mkdir hello-world-plus

Folder naming convention: Use lowercase letters and hyphens

2

Create Main Plugin File

Create a PHP file with the same name as your folder:

touch hello-world-plus/hello-world-plus.php

This will be the main entry point for your plugin

3

Add Plugin Header

Add the required plugin header information to identify your plugin to WordPress

Basic Plugin Code

hello-world-plus.php - Complete Basic Plugin

<?php
/**
 * Plugin Name: Hello World Plus
 * Plugin URI: https://example.com/plugins/hello-world-plus
 * Description: A simple plugin to demonstrate WordPress plugin development basics
 * Version: 1.0.0
 * Author: Your Name
 * Author URI: https://example.com
 * License: GPL v2 or later
 * License URI: https://www.gnu.org/licenses/gpl-2.0.html
 * Text Domain: hello-world-plus
 * Domain Path: /languages
 */

// Exit if accessed directly
if ( ! defined( 'ABSPATH' ) ) {
    exit;
}

// Define plugin constants
define( 'HWP_VERSION', '1.0.0' );
define( 'HWP_PLUGIN_DIR', plugin_dir_path( __FILE__ ) );
define( 'HWP_PLUGIN_URL', plugin_dir_url( __FILE__ ) );
define( 'HWP_PLUGIN_BASENAME', plugin_basename( __FILE__ ) );

/**
 * Main plugin class
 */
class Hello_World_Plus {
    
    /**
     * Instance of this class
     */
    private static $instance = null;
    
    /**
     * Get instance of this class (Singleton)
     */
    public static function get_instance() {
        if ( null === self::$instance ) {
            self::$instance = new self();
        }
        return self::$instance;
    }
    
    /**
     * Constructor
     */
    private function __construct() {
        $this->init_hooks();
    }
    
    /**
     * Initialize hooks
     */
    private function init_hooks() {
        // Activation/Deactivation hooks
        register_activation_hook( __FILE__, array( $this, 'activate' ) );
        register_deactivation_hook( __FILE__, array( $this, 'deactivate' ) );
        
        // Action hooks
        add_action( 'init', array( $this, 'init' ) );
        add_action( 'admin_menu', array( $this, 'add_admin_menu' ) );
        add_action( 'wp_enqueue_scripts', array( $this, 'enqueue_public_scripts' ) );
        add_action( 'admin_enqueue_scripts', array( $this, 'enqueue_admin_scripts' ) );
        
        // Filter hooks
        add_filter( 'the_content', array( $this, 'filter_content' ) );
        
        // Shortcode
        add_shortcode( 'hello_world', array( $this, 'hello_world_shortcode' ) );
    }
    
    /**
     * Plugin activation
     */
    public function activate() {
        // Create database tables if needed
        $this->create_tables();
        
        // Add default options
        add_option( 'hwp_message', 'Hello World!' );
        add_option( 'hwp_show_on_posts', true );
        
        // Flush rewrite rules
        flush_rewrite_rules();
    }
    
    /**
     * Plugin deactivation
     */
    public function deactivate() {
        // Clean up temporary data
        flush_rewrite_rules();
    }
    
    /**
     * Initialize plugin
     */
    public function init() {
        // Load text domain for translations
        load_plugin_textdomain( 
            'hello-world-plus', 
            false, 
            dirname( HWP_PLUGIN_BASENAME ) . '/languages' 
        );
    }
    
    /**
     * Add admin menu
     */
    public function add_admin_menu() {
        add_menu_page(
            __( 'Hello World Plus', 'hello-world-plus' ),           // Page title
            __( 'Hello World Plus', 'hello-world-plus' ),           // Menu title
            'manage_options',                                        // Capability
            'hello-world-plus',                                      // Menu slug
            array( $this, 'admin_page' ),                          // Callback
            'dashicons-smiley',                                     // Icon
            30                                                       // Position
        );
        
        // Add submenu
        add_submenu_page(
            'hello-world-plus',                                     // Parent slug
            __( 'Settings', 'hello-world-plus' ),                   // Page title
            __( 'Settings', 'hello-world-plus' ),                   // Menu title
            'manage_options',                                        // Capability
            'hello-world-plus-settings',                            // Menu slug
            array( $this, 'settings_page' )                        // Callback
        );
    }
    
    /**
     * Admin page content
     */
    public function admin_page() {
        ?>
        <div class="wrap">
            <h1><?php echo esc_html( get_admin_page_title() ); ?></h1>
            <p><?php _e( 'Welcome to Hello World Plus plugin!', 'hello-world-plus' ); ?></p>
            
            <div class="card">
                <h2><?php _e( 'Current Message', 'hello-world-plus' ); ?></h2>
                <p><strong><?php echo esc_html( get_option( 'hwp_message', 'Hello World!' ) ); ?></strong></p>
            </div>
            
            <div class="card">
                <h2><?php _e( 'How to Use', 'hello-world-plus' ); ?></h2>
                <p><?php _e( 'Use the shortcode [hello_world] to display your message anywhere.', 'hello-world-plus' ); ?></p>
                <p><?php _e( 'You can also pass a custom message:', 'hello-world-plus' ); ?> [hello_world message="Custom Message"]</p>
            </div>
        </div>
        <?php
    }
    
    /**
     * Settings page content
     */
    public function settings_page() {
        // Handle form submission
        if ( isset( $_POST['submit'] ) && check_admin_referer( 'hwp_settings_nonce' ) ) {
            update_option( 'hwp_message', sanitize_text_field( $_POST['hwp_message'] ) );
            update_option( 'hwp_show_on_posts', isset( $_POST['hwp_show_on_posts'] ) );
            echo '<div class="notice notice-success"><p>' . __( 'Settings saved!', 'hello-world-plus' ) . '</p></div>';
        }
        
        $message = get_option( 'hwp_message', 'Hello World!' );
        $show_on_posts = get_option( 'hwp_show_on_posts', true );
        ?>
        <div class="wrap">
            <h1><?php echo esc_html( get_admin_page_title() ); ?></h1>
            
            <form method="post" action="">
                <?php wp_nonce_field( 'hwp_settings_nonce' ); ?>
                
                <table class="form-table">
                    <tr>
                        <th scope="row">
                            <label for="hwp_message"><?php _e( 'Message', 'hello-world-plus' ); ?></label>
                        </th>
                        <td>
                            <input type="text" 
                                   id="hwp_message" 
                                   name="hwp_message" 
                                   value="<?php echo esc_attr( $message ); ?>" 
                                   class="regular-text" />
                            <p class="description">
                                <?php _e( 'Enter the message to display.', 'hello-world-plus' ); ?>
                            </p>
                        </td>
                    </tr>
                    <tr>
                        <th scope="row">
                            <?php _e( 'Display Options', 'hello-world-plus' ); ?>
                        </th>
                        <td>
                            <label>
                                <input type="checkbox" 
                                       name="hwp_show_on_posts" 
                                       value="1" 
                                       <?php checked( $show_on_posts, true ); ?> />
                                <?php _e( 'Automatically show on posts', 'hello-world-plus' ); ?>
                            </label>
                        </td>
                    </tr>
                </table>
                
                <?php submit_button(); ?>
            </form>
        </div>
        <?php
    }
    
    /**
     * Enqueue public scripts
     */
    public function enqueue_public_scripts() {
        wp_enqueue_style( 
            'hello-world-plus-public', 
            HWP_PLUGIN_URL . 'assets/css/public.css', 
            array(), 
            HWP_VERSION 
        );
        
        wp_enqueue_script( 
            'hello-world-plus-public', 
            HWP_PLUGIN_URL . 'assets/js/public.js', 
            array( 'jquery' ), 
            HWP_VERSION, 
            true 
        );
    }
    
    /**
     * Enqueue admin scripts
     */
    public function enqueue_admin_scripts( $hook ) {
        // Only load on our plugin pages
        if ( strpos( $hook, 'hello-world-plus' ) === false ) {
            return;
        }
        
        wp_enqueue_style( 
            'hello-world-plus-admin', 
            HWP_PLUGIN_URL . 'assets/css/admin.css', 
            array(), 
            HWP_VERSION 
        );
        
        wp_enqueue_script( 
            'hello-world-plus-admin', 
            HWP_PLUGIN_URL . 'assets/js/admin.js', 
            array( 'jquery' ), 
            HWP_VERSION, 
            true 
        );
    }
    
    /**
     * Filter content
     */
    public function filter_content( $content ) {
        if ( is_single() && get_option( 'hwp_show_on_posts', true ) ) {
            $message = get_option( 'hwp_message', 'Hello World!' );
            $content .= '<div class="hwp-message">' . esc_html( $message ) . '</div>';
        }
        return $content;
    }
    
    /**
     * Shortcode callback
     */
    public function hello_world_shortcode( $atts ) {
        $atts = shortcode_atts( array(
            'message' => get_option( 'hwp_message', 'Hello World!' ),
            'class' => 'hwp-shortcode'
        ), $atts, 'hello_world' );
        
        return '<div class="' . esc_attr( $atts['class'] ) . '">' . 
               esc_html( $atts['message'] ) . 
               '</div>';
    }
    
    /**
     * Create database tables
     */
    private function create_tables() {
        global $wpdb;
        
        $table_name = $wpdb->prefix . 'hello_world_messages';
        $charset_collate = $wpdb->get_charset_collate();
        
        $sql = "CREATE TABLE IF NOT EXISTS $table_name (
            id mediumint(9) NOT NULL AUTO_INCREMENT,
            message text NOT NULL,
            created_at datetime DEFAULT CURRENT_TIMESTAMP,
            PRIMARY KEY (id)
        ) $charset_collate;";
        
        require_once( ABSPATH . 'wp-admin/includes/upgrade.php' );
        dbDelta( $sql );
    }
}

// Initialize the plugin
function hwp_init() {
    return Hello_World_Plus::get_instance();
}
add_action( 'plugins_loaded', 'hwp_init' );

Testing Your Plugin

Test Checklist

  1. Activation: Go to Plugins page and activate your plugin
  2. Admin Menu: Check if menu appears in admin
  3. Settings Page: Test saving settings
  4. Shortcode: Add [hello_world] to a page/post
  5. Frontend Display: View a single post
  6. Deactivation: Deactivate and check for errors
  7. Debug Mode: Enable WP_DEBUG and check for notices

Common Issues and Solutions

Troubleshooting

Plugin doesn't appear in admin:

  • Check file permissions (should be 644 for files, 755 for folders)
  • Verify plugin header is correct
  • Check for PHP syntax errors

White screen of death:

  • Enable WP_DEBUG in wp-config.php
  • Check PHP error logs
  • Rename plugin folder via FTP to deactivate

Functions not working:

  • Verify hooks are properly registered
  • Check function names for typos
  • Ensure proper capability checks

Plugin Development Best Practices

  • Use OOP: Object-oriented programming for better organization
  • Singleton Pattern: Prevent multiple instances
  • Prefix Everything: Avoid naming conflicts
  • Escape Output: Always escape data before output
  • Sanitize Input: Clean all user input
  • Use Nonces: Verify form submissions
  • Check Capabilities: Ensure user permissions
  • Internationalization: Make plugin translatable
Never trust user input! Always sanitize, validate, and escape data appropriately.

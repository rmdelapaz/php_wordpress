Essential WordPress Hooks Overview

These 8 hooks form the foundation of most WordPress plugin development. Understanding them will cover 80% of your plugin development needs.

Hook Categories

Initialization: init, admin_init
Scripts/Styles: wp_enqueue_scripts, admin_enqueue_scripts
Content Filters: the_content, the_title
Post Operations: save_post
Query Modification: pre_get_posts

8. pre_get_posts ACTION

When It Fires:

Fires after the query variable object is created, but before the actual database query is run. Allows modification of queries.

Common Uses:

  • Modify main query parameters
  • Exclude categories from homepage
  • Change posts per page
  • Add custom sorting
  • Include custom post types in archives

pre_get_posts Hook Examples

<?php
// Modify main query
add_action( 'pre_get_posts', 'modify_main_query' );

function modify_main_query( $query ) {
    // Only modify main query on frontend
    if ( ! is_admin() && $query->is_main_query() ) {
        
        // Exclude category from homepage
        if ( $query->is_home() ) {
            $query->set( 'cat', '-5' ); // Exclude category ID 5
        }
        
        // Include custom post type in category archives
        if ( $query->is_category() ) {
            $query->set( 'post_type', array( 'post', 'product' ) );
        }
        
        // Change posts per page for search
        if ( $query->is_search() ) {
            $query->set( 'posts_per_page', 20 );
        }
        
        // Custom ordering for archive pages
        if ( $query->is_archive() ) {
            $query->set( 'orderby', 'menu_order' );
            $query->set( 'order', 'ASC' );
        }
    }
}

// Add meta query to filter posts
add_action( 'pre_get_posts', 'filter_posts_by_meta' );

function filter_posts_by_meta( $query ) {
    if ( ! is_admin() && $query->is_main_query() ) {
        if ( $query->is_post_type_archive( 'product' ) ) {
            $query->set( 'meta_query', array(
                array(
                    'key' => '_product_featured',
                    'value' => 'yes',
                    'compare' => '='
                )
            ) );
        }
    }
}

// Custom search modifications
add_action( 'pre_get_posts', 'enhance_search_query' );

function enhance_search_query( $query ) {
    if ( ! is_admin() && $query->is_search() && $query->is_main_query() ) {
        // Include multiple post types in search
        $query->set( 'post_type', array( 'post', 'page', 'product' ) );
        
        // Order by relevance
        $query->set( 'orderby', 'relevance' );
        
        // Exclude private posts
        $query->set( 'post_status', 'publish' );
    }
}
Always check is_main_query() to avoid affecting secondary queries like widgets or navigation menus!

Quick Reference Summary

Hook Name Type When to Use Priority
init Action Register post types, taxonomies, shortcodes 10 (default)
admin_init Action Admin-only initialization, settings registration 10 (default)
wp_enqueue_scripts Action Enqueue frontend CSS/JS 10 (default)

Combining Multiple Hooks

Complete Plugin Example Using All 8 Hooks

<?php
/**
 * Plugin Name: Hook Master
 * Description: Demonstrates all 8 common hooks
 */

class Hook_Master_Plugin {
    
    public function __construct() {
        // Initialization hooks
        add_action( 'init', array( $this, 'init_hook' ) );
        add_action( 'admin_init', array( $this, 'admin_init_hook' ) );
        
        // Script/Style hooks
        add_action( 'wp_enqueue_scripts', array( $this, 'frontend_scripts' ) );
        add_action( 'admin_enqueue_scripts', array( $this, 'admin_scripts' ) );
        
        // Content filter hooks
        add_filter( 'the_content', array( $this, 'filter_content' ) );
        add_filter( 'the_title', array( $this, 'filter_title' ), 10, 2 );
        
        // Post operation hooks
        add_action( 'save_post', array( $this, 'save_post_data' ), 10, 3 );
        
        // Query hooks
        add_action( 'pre_get_posts', array( $this, 'modify_queries' ) );
    }
    
    public function init_hook() {
        // Register custom post type
        register_post_type( 'portfolio', array(
            'public' => true,
            'label' => 'Portfolio'
        ) );
    }
    
    public function admin_init_hook() {
        // Register settings
        register_setting( 'hook_master_group', 'hook_master_options' );
    }
    
    public function frontend_scripts() {
        wp_enqueue_style( 'hook-master', plugin_dir_url( __FILE__ ) . 'style.css' );
    }
    
    public function admin_scripts( $hook ) {
        if ( 'settings_page_hook-master' === $hook ) {
            wp_enqueue_script( 'hook-master-admin', plugin_dir_url( __FILE__ ) . 'admin.js' );
        }
    }
    
    public function filter_content( $content ) {
        if ( is_single() ) {
            $content .= '
Added by Hook Master
';
        }
        return $content;
    }
    
    public function filter_title( $title, $id ) {
        if ( get_post_type( $id ) === 'portfolio' ) {
            $title = '‚òÖ ' . $title;
        }
        return $title;
    }
    
    public function save_post_data( $post_id, $post, $update ) {
        if ( 'portfolio' === $post->post_type ) {
            update_post_meta( $post_id, '_last_saved', time() );
        }
    }
    
    public function modify_queries( $query ) {
        if ( ! is_admin() && $query->is_main_query() && $query->is_home() ) {
            $query->set( 'post_type', array( 'post', 'portfolio' ) );
        }
    }
}

// Initialize plugin
new Hook_Master_Plugin();

Best Practices for Common Hooks

Always check context (is_admin(), is_main_query(), etc.) before modifying behavior to avoid unintended side effects.
  • Performance: Keep init and admin_init lightweight
  • Context awareness: Check where hooks are firing
  • Return values: Always return in filter hooks
  • Security: Verify nonces and capabilities in save_post
  • Dependencies: Check if functions exist before using
  • Debugging: Use error_log() to debug hooks
  • Priority: Use appropriate priorities for execution order
  • Documentation: Comment why you're using each hook

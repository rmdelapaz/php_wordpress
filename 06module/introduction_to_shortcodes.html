What Are Shortcodes?

Shortcodes are WordPress-specific code snippets that let you do complex things with very little effort. They can embed files or create objects that would normally require lots of complicated code in just one line.

Creating Your First Shortcode

Basic Shortcode Structure

// Step 1: Create the callback function
function my_first_shortcode() {
    return 'Hello, World! This is my first shortcode.';
}

// Step 2: Register the shortcode
add_shortcode( 'hello', 'my_first_shortcode' );

// Usage in content: [hello]
Important: Shortcode functions must RETURN their output, not echo it. This is a common mistake that can break page layouts.

More Practical Example

// Display current year - useful for copyright notices
function current_year_shortcode() {
    return date('Y');
}
add_shortcode( 'year', 'current_year_shortcode' );

// Usage: Copyright ¬© [year] Your Company

// Display user information
function user_info_shortcode() {
    if ( is_user_logged_in() ) {
        $current_user = wp_get_current_user();
        return 'Welcome, ' . esc_html( $current_user->display_name ) . '!';
    } else {
        return 'Welcome, Guest! Please log in.';
    }
}
add_shortcode( 'user_info', 'user_info_shortcode' );

// Display recent posts
function recent_posts_shortcode() {
    $query = new WP_Query( array(
        'posts_per_page' => 5,
        'post_status' => 'publish'
    ) );
    
    if ( ! $query->have_posts() ) {
        return 'No posts found.';
    }
    
    $output = '
    ';
    
    while ( $query->have_posts() ) {
        $query->the_post();
        $output .= '
  • ' . get_the_title() . '
    • ';
    }
    
    $output .= '
';
    
    wp_reset_postdata();
    
    return $output;
}
add_shortcode( 'recent_posts', 'recent_posts_shortcode' );

Anatomy of a Shortcode Function

Shortcode Function Components

Function Name Should be unique and descriptive, often prefixed to avoid conflicts
Parameters Can accept $atts (attributes), $content (enclosed content), and $tag (shortcode name)
Processing Logic Your PHP code that generates the output
Return Statement Must return (not echo) the generated HTML/content 
function prefix_shortcode_name( $atts = array(), $content = null, $tag = '' ) {
    // 1. Parse attributes with defaults
    $atts = shortcode_atts( array(
        'attribute1' => 'default1',
        'attribute2' => 'default2',
    ), $atts, $tag );
    
    // 2. Process the shortcode logic
    $output = '
';
    $output .= '
Attribute 1: ' . esc_html( $atts['attribute1'] ) . '
';
    
    if ( ! is_null( $content ) ) {
        $output .= '
' . do_shortcode( $content ) . '
';
    }
    
    $output .= '
';
    
    // 3. Always return, never echo
    return $output;
}

add_shortcode( 'my_shortcode', 'prefix_shortcode_name' );

Enclosing Shortcodes

Working with Content Between Tags

// Create a highlight box shortcode
function highlight_box_shortcode( $atts, $content = null ) {
    // Parse attributes
    $atts = shortcode_atts( array(
        'color' => 'yellow',
        'style' => 'normal', // normal, important, warning
    ), $atts );
    
    // Don't process if no content
    if ( is_null( $content ) ) {
        return '';
    }
    
    // Build the output
    $classes = 'highlight-box highlight-' . esc_attr( $atts['style'] );
    $style = 'background-color: ' . esc_attr( $atts['color'] ) . ';';
    
    $output = '
';
    $output .= '
';
    // Process nested shortcodes in content
    $output .= do_shortcode( $content );
    $output .= '
';
    $output .= '
';
    
    return $output;
}
add_shortcode( 'highlight', 'highlight_box_shortcode' );

// Usage: [highlight color="#ffe4b5" style="important"]This is important text![/highlight]

// Alert box shortcode
function alert_shortcode( $atts, $content = null ) {
    $atts = shortcode_atts( array(
        'type' => 'info', // info, success, warning, error
        'dismissible' => 'no',
        'icon' => 'yes',
    ), $atts );
    
    if ( is_null( $content ) ) {
        return '';
    }
    
    $classes = 'alert alert-' . esc_attr( $atts['type'] );
    if ( $atts['dismissible'] === 'yes' ) {
        $classes .= ' alert-dismissible';
    }
    
    $output = '
';
    
    if ( $atts['dismissible'] === 'yes' ) {
        $output .= '';
    }
    
    if ( $atts['icon'] === 'yes' ) {
        $icon = '';
        switch ( $atts['type'] ) {
            case 'success':
                $icon = '‚úì';
                break;
            case 'warning':
                $icon = '‚ö†';
                break;
            case 'error':
                $icon = '‚úó';
                break;
            default:
                $icon = '‚Ñπ';
        }
        $output .= '' . $icon . ' ';
    }
    
    $output .= wpautop( do_shortcode( $content ) );
    $output .= '
';
    
    return $output;
}
add_shortcode( 'alert', 'alert_shortcode' );

// Usage: [alert type="warning" dismissible="yes"]Warning message here![/alert]

Common Shortcode Patterns

Button Shortcode

function button_shortcode( $atts, $content = null ) {
                                    $atts = shortcode_atts( array(
        'url' => '#',
        'target' => '_self',
        'color' => 'primary',
        'size' => 'medium',
        'icon' => '',
        'class' => '',
    ), $atts );
    
    $classes = 'btn btn-' . esc_attr( $atts['color'] ) . ' btn-' . esc_attr( $atts['size'] );
    if ( ! empty( $atts['class'] ) ) {
        $classes .= ' ' . esc_attr( $atts['class'] );
    }
    
    $output = '';
    
    if ( ! empty( $atts['icon'] ) ) {
        $output .= ' ';
    }
    
    $output .= do_shortcode( $content );
    $output .= '';
    
    return $output;
}
add_shortcode( 'button', 'button_shortcode' );

// Usage: [button url="/contact" color="success" size="large"]Contact Us[/button]

Columns Layout Shortcode

// Row shortcode
function row_shortcode( $atts, $content = null ) {
    $atts = shortcode_atts( array(
        'class' => '',
    ), $atts );
    
    $classes = 'row';
    if ( ! empty( $atts['class'] ) ) {
        $classes .= ' ' . esc_attr( $atts['class'] );
    }
    
    return '
' . do_shortcode( $content ) . '
';
}
add_shortcode( 'row', 'row_shortcode' );

// Column shortcode
function column_shortcode( $atts, $content = null ) {
    $atts = shortcode_atts( array(
        'width' => '1/2',
        'class' => '',
    ), $atts );
    
    // Convert fractions to grid classes
    $width_map = array(
        '1/2' => 'col-md-6',
        '1/3' => 'col-md-4',
        '2/3' => 'col-md-8',
        '1/4' => 'col-md-3',
        '3/4' => 'col-md-9',
        '1/6' => 'col-md-2',
        '5/6' => 'col-md-10',
        'full' => 'col-md-12',
    );
    
    $classes = isset( $width_map[$atts['width']] ) ? $width_map[$atts['width']] : 'col-md-6';
    
    if ( ! empty( $atts['class'] ) ) {
        $classes .= ' ' . esc_attr( $atts['class'] );
    }
    
    return '
' . do_shortcode( $content ) . '
';
}
add_shortcode( 'column', 'column_shortcode' );

// Usage:
// [row]
//   [column width="1/3"]First column content[/column]
//   [column width="2/3"]Second column content[/column]
// [/row]

Shortcode Best Practices

‚úÖ Do's

  • Always return output - Never echo directly in shortcode functions
  • Use unique names - Prefix your shortcodes to avoid conflicts
  • Sanitize and escape - Always sanitize attributes and escape output
  • Provide defaults - Use shortcode_atts() for default values
  • Process nested shortcodes - Use do_shortcode() on content
  • Check for null content - Handle cases where content might be empty
  • Document your shortcodes - Provide clear usage examples

‚ùå Don'ts

  • Don't echo output - This will appear at the wrong place on the page
  • Don't use generic names - Avoid names like [button] without prefix
  • Don't trust user input - Always sanitize and validate
  • Don't create overly complex shortcodes - Keep them simple and focused
  • Don't forget wpautop() - When dealing with paragraph content
  • Don't use extract() - Access attributes directly from array

Debugging Shortcodes

Common Issues and Solutions

// Debug helper function
function debug_shortcode( $atts, $content = null, $tag = '' ) {
    // Start output buffering to capture any errors
    ob_start();
    
    echo '
';
    echo 'Shortcode Debug Info' . "\n";
    echo 'Tag: ' . esc_html( $tag ) . "\n";
    echo 'Attributes: ';
    print_r( $atts );
    echo 'Content: ' . esc_html( $content ) . "\n";
    echo 'Is Admin: ' . ( is_admin() ? 'Yes' : 'No' ) . "\n";
    echo 'Current User Can Edit: ' . ( current_user_can( 'edit_posts' ) ? 'Yes' : 'No' ) . "\n";
    echo '
';
    
    return ob_get_clean();
}

// Only show debug info to administrators
function conditional_debug_shortcode( $atts, $content = null, $tag = '' ) {
    if ( current_user_can( 'manage_options' ) ) {
        return debug_shortcode( $atts, $content, $tag );
    }
    return '';
}

add_shortcode( 'debug', 'conditional_debug_shortcode' );

Debugging Tips:

  • Use error_log() to log debug information
  • Check if shortcode is registered with shortcode_exists()
  • Use output buffering for complex HTML generation
  • Test with different user roles and permissions
  • Check for PHP errors in debug.log
  • Use browser developer tools to inspect generated HTML

üìã Detailed Requirements

1. Plugin Foundation (20 points)

  • Create main plugin file with proper header
  • Implement activation/deactivation hooks
  • Set up proper file structure and autoloading
  • Include uninstall functionality
  • Add text domain for internationalization

2. Portfolio Custom Post Type (20 points)

  • Register 'portfolio' custom post type
  • Set appropriate labels and arguments
  • Enable REST API support
  • Add custom icon and menu position
  • Configure proper capabilities
  • Set up permalink structure

3. Taxonomies (15 points)

  • Project Type (Hierarchical): Website, Mobile App, Branding, etc.
  • Technologies (Non-Hierarchical): WordPress, React, PHP, etc.
  • Client Industry (Hierarchical): Healthcare, Education, E-commerce, etc.
  • All taxonomies should have proper labels and be shown in admin columns

4. Custom Fields & Meta Boxes (20 points)

  • Project Details Meta Box:
    • Client Name (text)
    • Project URL (url)
    • Project Date (date picker)
    • Project Status (select: In Progress, Completed, Archived)
  • Project Gallery Meta Box:
    • Featured Image (already exists)
    • Gallery Images (multiple image upload)
  • Testimonial Meta Box:
    • Client Testimonial (textarea)
    • Client Name (text)
    • Client Position (text)
  • All fields must be properly saved with security checks

5. Templates (15 points)

  • Single Portfolio Template:
    • Display all project information
    • Show image gallery
    • Display client testimonial
    • Show related projects
  • Portfolio Archive Template:
    • Grid layout with featured images
    • Filtering by taxonomies
    • Pagination
  • Taxonomy Templates:
    • Custom templates for each taxonomy

6. Shortcode (10 points)

  • Create [portfolio] shortcode with parameters:
  • count (number of items)
  • type (project type filter)
  • columns (layout columns)
  • orderby (date, title, menu_order)
  • Example: [portfolio count="6" type="website" columns="3"]

üöÄ Starter Code

Main Plugin File (portfolio-cpt-plugin.php)

<?php
/**
 * Plugin Name: Portfolio CPT Plugin
 * Plugin URI: https://yourwebsite.com/
 * Description: A complete portfolio management system with custom post types
 * Version: 1.0.0
 * Author: Your Name
 * Author URI: https://yourwebsite.com/
 * License: GPL v2 or later
 * Text Domain: portfolio-cpt-plugin
 * Domain Path: /languages
 */

// Prevent direct access
if ( ! defined( 'ABSPATH' ) ) {
    exit;
}

// Define plugin constants
define( 'PORTFOLIO_CPT_VERSION', '1.0.0' );
define( 'PORTFOLIO_CPT_PATH', plugin_dir_path( __FILE__ ) );
define( 'PORTFOLIO_CPT_URL', plugin_dir_url( __FILE__ ) );

// Activation hook
register_activation_hook( __FILE__, 'portfolio_cpt_activate' );
function portfolio_cpt_activate() {
    // Flush rewrite rules
    flush_rewrite_rules();
    
    // Set default options
    add_option( 'portfolio_cpt_version', PORTFOLIO_CPT_VERSION );
}

// Deactivation hook
register_deactivation_hook( __FILE__, 'portfolio_cpt_deactivate' );
function portfolio_cpt_deactivate() {
    // Flush rewrite rules
    flush_rewrite_rules();
}

// Load plugin text domain
add_action( 'plugins_loaded', 'portfolio_cpt_load_textdomain' );
function portfolio_cpt_load_textdomain() {
    load_plugin_textdomain( 
        'portfolio-cpt-plugin', 
        false, 
        dirname( plugin_basename( __FILE__ ) ) . '/languages' 
    );
}

// Include required files
require_once PORTFOLIO_CPT_PATH . 'includes/class-portfolio-cpt.php';
require_once PORTFOLIO_CPT_PATH . 'includes/class-portfolio-taxonomies.php';
require_once PORTFOLIO_CPT_PATH . 'includes/class-portfolio-metaboxes.php';
require_once PORTFOLIO_CPT_PATH . 'includes/class-portfolio-shortcode.php';
require_once PORTFOLIO_CPT_PATH . 'includes/class-portfolio-templates.php';

// Initialize classes
new Portfolio_CPT();
new Portfolio_Taxonomies();
new Portfolio_Metaboxes();
new Portfolio_Shortcode();
new Portfolio_Templates();

CPT Class Starter (includes/class-portfolio-cpt.php)

<?php
class Portfolio_CPT {
    
    public function __construct() {
        add_action( 'init', array( $this, 'register_portfolio_cpt' ) );
        add_action( 'admin_menu', array( $this, 'add_admin_menu' ) );
    }
    
    public function register_portfolio_cpt() {
        $labels = array(
            'name'                  => _x( 'Portfolio', 'Post type general name', 'portfolio-cpt-plugin' ),
            'singular_name'         => _x( 'Portfolio Item', 'Post type singular name', 'portfolio-cpt-plugin' ),
            // Add all other labels...
        );
        
        $args = array(
            'label'                 => __( 'Portfolio', 'portfolio-cpt-plugin' ),
            'labels'                => $labels,
            'supports'              => array( 'title', 'editor', 'thumbnail', 'excerpt', 'revisions' ),
            'public'                => true,
            'show_ui'               => true,
            'show_in_menu'          => true,
            'menu_position'         => 20,
            'menu_icon'             => 'dashicons-portfolio',
            'show_in_admin_bar'     => true,
            'show_in_nav_menus'     => true,
            'can_export'            => true,
            'has_archive'           => 'portfolio',
            'rewrite'               => array( 'slug' => 'portfolio' ),
            'show_in_rest'          => true,
            // Add other arguments...
        );
        
        register_post_type( 'portfolio', $args );
    }
    
    public function add_admin_menu() {
        // Add any submenu pages if needed
    }
}

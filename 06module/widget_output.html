The widget() Method

The widget() method is responsible for displaying your widget's content on the front-end. It receives display arguments from the theme and the saved widget settings.

Argument Description Example
$before_widget HTML to display before widget <aside id="widget-1" class="widget">
$after_widget HTML to display after widget </aside>
$before_title HTML before widget title <h3 class="widget-title">
$after_title HTML after widget title </h3>
$widget_id Unique widget instance ID my_widget-2
$widget_name Widget class name My_Custom_Widget

Basic Widget Output Structure

public function widget( $args, $instance ) {
    // Extract widget display arguments
    extract( $args );
    
    // Get widget settings with defaults
    $title = ! empty( $instance['title'] ) 
        ? apply_filters( 'widget_title', $instance['title'], $instance, $this->id_base ) 
        : '';
    
    // Start widget output
    echo $before_widget;
    
    // Display title if present
    if ( ! empty( $title ) ) {
        echo $before_title . esc_html( $title ) . $after_title;
    }
    
    // Widget content area
    ?>

Adding JavaScript Functionality

Widget JavaScript (portfolio-widget.js)

jQuery(document).ready(function($) {
    
    // Portfolio Filter Functionality
    $('.portfolio-filters .filter-btn').on('click', function() {
        var $this = $(this);
        var filter = $this.data('filter');
        var $container = $this.closest('.portfolio-widget');
        var $items = $container.find('.portfolio-item');
        
        // Update active button
        $this.siblings().removeClass('active');
        $this.addClass('active');
        
        // Filter items
        if (filter === 'all') {
            $items.removeClass('filtered-out').addClass('filtered-in');
        } else {
            $items.each(function() {
                var $item = $(this);
                if ($item.hasClass('category-' + filter)) {
                    $item.removeClass('filtered-out').addClass('filtered-in');
                } else {
                    $item.removeClass('filtered-in').addClass('filtered-out');
                }
            });
        }
    });
    
    // Carousel Functionality
    $('.portfolio-style-carousel').each(function() {
        var $carousel = $(this);
        var $items = $carousel.find('.portfolio-item');
        var $dots = $carousel.find('.carousel-dot');
        var currentIndex = 0;
        var itemsPerSlide = 3; // Adjust based on columns
        
        function showSlide(index) {
            $items.removeClass('active');
            $dots.removeClass('active');
            
            var start = index * itemsPerSlide;
            var end = start + itemsPerSlide;
            
            $items.slice(start, end).addClass('active');
            $dots.eq(index).addClass('active');
            
            currentIndex = index;
        }
        
        // Initialize
        showSlide(0);
        
        // Next button
        $carousel.find('.carousel-next').on('click', function() {
            var totalSlides = Math.ceil($items.length / itemsPerSlide);
            var nextIndex = (currentIndex + 1) % totalSlides;
            showSlide(nextIndex);
        });
        
        // Previous button
        $carousel.find('.carousel-prev').on('click', function() {
            var totalSlides = Math.ceil($items.length / itemsPerSlide);
            var prevIndex = currentIndex - 1;
            if (prevIndex < 0) prevIndex = totalSlides - 1;
            showSlide(prevIndex);
        });
        
        // Dot navigation
        $dots.on('click', function() {
            showSlide($(this).data('slide'));
        });
        
        // Auto-play (optional)
        var autoPlay = $carousel.data('autoplay');
        if (autoPlay) {
            setInterval(function() {
                $carousel.find('.carousel-next').click();
            }, 5000);
        }
    });
    
    // Lightbox Integration (using a library like Magnific Popup)
    if ($.fn.magnificPopup) {
        $('.portfolio-lightbox').magnificPopup({
            type: 'image',
            gallery: {
                enabled: true,
                navigateByImgClick: true,
                preload: [0, 1]
            },
            image: {
                titleSrc: function(item) {
                    return item.el.attr('data-title');
                }
            }
        });
    }
    
    // AJAX Load More
    $('.portfolio-load-more').on('click', function(e) {
        e.preventDefault();
        
        var $button = $(this);
        var $container = $button.closest('.portfolio-widget').find('.portfolio-grid');
        var page = $button.data('page') || 1;
        
        $button.text('Loading...').prop('disabled', true);
        
        $.ajax({
            url: portfolioWidget.ajaxurl,
            type: 'POST',
            data: {
                action: 'load_more_portfolio',
                page: page + 1,
                nonce: portfolioWidget.nonce
            },
            success: function(response) {
                if (response.success) {
                    $container.append(response.data.html);
                    $button.data('page', page + 1);
                    
                    if (!response.data.has_more) {
                        $button.hide();
                    } else {
                        $button.text('Load More').prop('disabled', false);
                    }
                }
            },
            error: function() {
                $button.text('Error loading').prop('disabled', false);
            }
        });
    });
    
    // Masonry Layout (if enabled)
    if ($.fn.masonry) {
        $('.portfolio-style-masonry .portfolio-grid').masonry({
            itemSelector: '.portfolio-item',
            columnWidth: '.portfolio-item',
            percentPosition: true
        });
    }
    
    // Responsive adjustments
    function adjustPortfolioLayout() {
        var windowWidth = $(window).width();
        
        $('.portfolio-widget').each(function() {
            var $widget = $(this);
            var columns = $widget.data('columns') || 3;
            
            if (windowWidth < 480) {
                columns = 1;
            } else if (windowWidth < 768) {
                columns = Math.min(2, columns);
            }
            
            $widget.find('.portfolio-grid').css({
                'grid-template-columns': 'repeat(' + columns + ', 1fr)'
            });
        });
    }
    
    $(window).on('resize', adjustPortfolioLayout);
    adjustPortfolioLayout();
    
});

Output & Styling Best Practices

✅ Do's

  • Use semantic HTML5 elements
  • Implement responsive design from the start
  • Use CSS classes instead of inline styles
  • Enqueue scripts and styles properly
  • Cache expensive operations
  • Implement lazy loading for images
  • Use CSS custom properties for theming
  • Test across different themes
  • Provide accessibility features

❌ Don'ts

  • Don't hardcode colors and dimensions
  • Don't use !important excessively
  • Don't forget about RTL support
  • Don't load resources on every page
  • Don't ignore theme styles completely
  • Don't use deprecated HTML tags
  • Don't forget ARIA labels
